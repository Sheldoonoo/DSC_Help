Saving a Sequence to File 

To save the current sequence to file for future recall:

  1. Access either the Design View or Running Queue, depending on which sequence you wish to save.
  2. Click Save Sequence .
  3. Locate folder where you wish to save your file.
  4. Enter the name for the file.
  5. Click Save.

Opening a Saved Sequence File

To open a saved sequence file:

  1. Access either the Design View or Running Queue, depending upon where you wish to open the file. The current displayed sequence is replaced with the new one.
  2. Click Open Sequence .
  3. Locate the desired folder containing the file of interest. Select a file from the list.
  4. Click Open.
