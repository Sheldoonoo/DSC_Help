Shutting Down the Discovery Instrument

Occasionally, it may be necessary to shut down (to clear an error or restart after a firmware download update) the Discovery DSC or Discovery TGA system.

Before powering down the system, consider the following:

  • All of the thermal analysis system components are designed to be powered on for extended periods of time.
  • The electronics of the instrument and the controller perform more reliably if power fluctuations (caused by turning units on and off) are minimized.

For these reasons, turning the system and its components on and off frequently is discouraged. Therefore, when you are finished running an experiment on the instrument and wish to use the thermal analysis system for some other task, it is recommended that the instrument remain on.

Shutting Down the Discovery Instrument

  1. NOTE: To ensure proper shutdown of the instrument, it is recommended that you Shutdown the instrument before powering it off.
  1. You can initiate an system shutdown through either TRIOS software or through the instrument user interface. Select from one of the options below.
  • From the Discovery instrument's user interface main menu, access Utilities > Shutdown.
  • From TRIOS software, access the Instrument menu option and select Shutdown (Instrument).

 

  1. A confirmation appears on the user interface screen or in TRIOS software (TRIOS software confirmation message shown below). Press or click Yes to continue. All communication halts while the system goes through the shutdown procedure.

  1. NOTE: To ensure proper shutdown of the instrument, it is recommended that you wait until the user interface screen is black, before setting the power switch to OFF (0).
