Modifying a Running Discovery Instrument Method

The Running Method List is available when the instrument is in a Running state within the Control panel. If the Control panel is currently not displayed, click the Experiment tab > Controls .

This panel displays the current method segments for the active experiment. The check mark indicates that the method segment has been successfully completed and the green ball signifies the current running method segment.

Modify the Running Method

If you would like to modify the running method, click the Editing button in the Running Method List within the Control panel.

You will then be able to edit the current method and any future method segments.

Click Apply to apply your changes.

Add a Segment

To add a segment click on the add button and either double click on the method segment to append the segment at the end of the method or click the segment and drag it into the list.

Click Apply to apply your changes.

Advance the Active Method

If you would like to advance the active method to the next segment, click Go to Next Segment button. The method automatically advances to the next segment. If this option was selected while the last segment was active, the method ends.
