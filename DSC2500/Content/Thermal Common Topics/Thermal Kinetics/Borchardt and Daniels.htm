Overview

The Borchardt and Daniels (B/D) kinetics approach permits the calculation of activation energy ( Eα), pre-exponential factor (Z), heat of reaction (H), reaction order (n), and rate constant (k) from a single DSC scan. This approach was originally described by Borchardt and Daniels [1] for solutions and was subsequently refined for solids by other researchers [2]. Eα= Activation energy (J/mol)

Z = pre-exponential factor or Arrhenius frequency factor (1/sec)

R = gas constant = 8.314 J/mol K

Equation (4) can be solved with a multiple linear regression of the general form: z = a + bx +cy where z = ln[dα/dt], a = ln[Z], b = Eα/R, x = 1/T, c = n, and y = ln[1 – α]. The parameters dα/dt, T, and α are determined from the DSC exotherm [2]. The determination uses a specified number of segments (default = 20) of the curve, evenly spaced by temperature with heat flow (dH/dt), and absolute temperature (T) being determined at each point. The default conditions are that the first segment starts at 10 percent of peak height and the last segment ends at 90 percent of peak area but these segment ranges should be user adjustable. Initially the whole of the exotherm is integrated as a function of time to give total energy of the reaction (H). Then the individual heat flow measurements (dH/dt) are normalized to the total energy to give dα/dt = (dH/dt)/H. The amount left to react (α) is that energy fraction of the exotherm after the indicated temperature (Hf) normalized by the total energy α = Hf/H. Thus the value of (1 – α) goes from 0 to 1 during the course of the reaction. Once the value of the reaction order (n) has been determined by multiple linear regression, then the value of ln [k(T)] may be determined by ln[k(T)] = ln [(dα/dt)/(1 – α)n].

Experimental

The experimental details to perform a B&D kinetics analysis are very straightforward. The user performs a single heating experiment, typically at 10°C/min. The resultant thermal curve will generally include a broad exothermic peak corresponding to the kinetic process under investigation. This peak is integrated using DSC Conversion Calculation as shown in the example below.

Analysis

Once the data is plotted, it is necessary to analyze the data using DSC Conversion Calculation prior to generating the kinetics:

  • NOTE: Contact TA Instruments Sales to acquire the Kinetics key.
  1. From the Graph > Analysis menu, select DSC Conversion Calculation.
  2. Select the peak integration limits, and then click Analyze. A new document tab is generated with the conversion data.
  3. With the conversion ramp plot displayed, select Borchardt and Daniels nth Order from the Analysis list and click Analyze.
  4. Click Edit Analysis to verify selections.

  1. Click Accept. A new document tab displays for the Rate Constant Plot (a Half Life plot may also be obtained from this document).

Additional Plots

Two predictive curves are obtained from the B&D kinetic modeling: isothermal plots and isoconversion plots. Isothermal plots provide time conditions and degree of conversion information for specific isothermal temperatures. Isoconversion plots provide time and temperature conditions for a specific conversion level. Both predictive plots are useful in the development and control of reaction conditions in order to achieve the desired final product.

  1. Display the conversion plot.
  2. From the File Manager, select the conversion plot and then right-click and select Transformations... > Kinetics Conversion...

  1. A window to define the desired conversion plot as well as the parameters of that plot displays.

  1. In the Conversion Type drop-down menu, select either Isothermal or IsoConversion.
  2. Select either a single temperature or temperature range.
  3. Enter in the desired temperature.
  4. Select Show Options to specify the number of points for each curve as well as the data spacing (log or linear).
  1. A new document is created for this plot.

Analysis Settings

  1. Enter the conversion limits over which the kinetics results will be determined.
  2. Enter the desired number of temperature intervals (default = 20).
  3. Select the Calculation type.
    1. Select Calculate and the program will calculate and display the reaction order when the analysis is performed
    2. Select Fixed if you want to enter your own reaction order. Based upon kinetic theory, the chemical reaction order should be a small integer (commonly 1). Enter the reaction order desired for your particular kinetics reaction. The program will other kinetics parameters based on this reaction order.
  4. Click Accept.
