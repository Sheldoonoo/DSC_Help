Creating a Baseline

The Baseline feature allows you to transform one open graph by adding y-coordinate values from another open graph. Follow the steps below to either subtract or add the baseline.

  1. Open the baseline reference file.
  2. Open the desired sample file and select the desired signals that will be used to perform operations with the baseline reference file.
  3. Make sure that the sample data file window is currently active. Right-click on the graph and select Baseline from the menu.

  1. The Baseline dialog box displays. Select the desired reference file to be used as the baseline from the available list.

  1. Check the desired signal(s) from those shown in the Variables box.
  2. Click on the desired Base: Time or Temperature.
  3. Click on the desired operation: Add or Subtract the baseline file.
  4. Click on the desired signal match: Name and units or Units only.
  5. Select Limit data to the range of the reference if the baseline file has a smaller range than the sample file, otherwise the resulting graph will be incomplete.
  6. Click OK. The resulting data is displayed.
  • NOTE: A baseline signal will be added to or subtracted from each data file signal that has the same units and/or signal name. Those signals that are not compatible between the files are not used.
  • NOTE: The baseline addition/subtraction is only applied to y-axis signals. It is not applied to x-axis signals. If, for example, you wish to subtract the temperature signal, plot temperature on the y-axis instead of the x-axis, then perform the baseline subtraction.
