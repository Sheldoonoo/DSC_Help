Creating a Data Table

The Data Table feature allows you to create a spreadsheet on a select range of data, using either all data points or a selected increment.

To create the Data Table:

  1. Right-click the graph and then click Data Table.
  2. The Create Data Table dialog box displays.

  • In the Start field, enter the x coordinate in which the data range should start. In the Stop field, enter the x coordinate in which the data should stop.
  • Choose how the data will be represented in the spreadsheet:
    • The Increment By field interpolates data between measured data points.
    • Select All Data Points to have all data points added to the spreadsheet.
    • If Time and Temperature are not already being included in the data, you can choose to include them by selecting Time and Temperature.
    • Select Valid Data Points Only to keep any incorrect data from being included in the Data Table.

Viewing the Data Table

Once the Data Table is created, you can review the collected data by procedure step by doing one of the following:

  • Choose the step from the Selected step drop-down menu at the top of the table.
  • Highlight the step from the Results pane. Right-click and then select Send to Spreadsheet or Send to New Spreadsheet. A next to the step in the Results pane indicates that it is currently in the Data Table. Select Remove from Spreadsheet from the right-click menu to remove the step from the Data Table.

Editing the Data Table

You can add or remove a row in the spreadsheet by highlighting the row, right-clicking and selecting Add Row or Delete Row. Additionally from the right-click menu, you can Append, Hide and Unhide rows, add comments to fields, and mark data as invalid.
