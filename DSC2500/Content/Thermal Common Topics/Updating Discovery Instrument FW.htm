Updating Discovery Instrument Firmware using TRIOS Software

To take advantage of new features in the TRIOS software, it may be necessary to update the instrument firmware. Follow these instructions to complete the firmware update process:

  1. Install the new TRIOS software on your PC, which can be downloaded from the TA Instruments web site > Support tab > Software Downloads and Support.
  2. Turn on the instrument.
  3. Open TRIOS software. If a message displays indicating a firmware update is needed, follow the on-screen instructions. Otherwise, proceed to the next step.
  4. Click the TRIOS button at the top-left corner of the application.
  5. Select Options from the drop-down menu.
  6. Access the instrument options (for example, Discovery DSC > Information) from the left-hand panel. The following window displays (Discovery DSC models 25/250/2500 shown).

  1. Click Update. A Firmware Install window displays. Click Install. The necessary files required to update your instrument are sent to the instrument. When the update finishes, the below window displays. Click Close and then click OK in the Options dialog.

  1. NOTE: If your computer requires proxy authentication to access LAN connections, then you will receive the following message when you display the Instrument Update window.

To correct this error, access your computer's Internet Properties. Click the Connections tab, then select LAN Settings. Within these options, uncheck Automatically detect settings, Use automatic configuration script and Use a proxy server for your LAN, if these are selected.

Click OK and then OK again to close these options. Once you complete the download process, return to these options and re-enable your settings.

  1. Firmware update is complete. To view the updated firmware versions, return to the Discovery (instrument) > Information window within the TRIOS Options or the System Info > About window from the instrument user interface.
