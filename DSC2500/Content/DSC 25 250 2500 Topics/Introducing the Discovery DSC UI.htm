The Discovery DSC uses an integrated user interface display for local operator control. The functions of the user interface change depending upon the view displayed. This section briefly describes the basic layout of these functions.

The instrument user interface has three basic sections:

  • The status line along the top of the display indicates the current instrument status and sample temperature.
  • The view panel allows for real-time instrument status and associated actions.
  • The primary function buttons allow for easy access of common functions.

View Panel

The view panel provides real-time instrument status and additional functionality pertinent to the selected operation. A list of available functions is described below.

 

View Panel Functions on the User Interface
Button Description

System Info

Displays instrument information such as the serial number, IP configuration, and network configuration.

Signals

User can select signals to display the real-time signal data generated directly from the instrument.

Settings

Displays options for manual controls, such as Event, Air Cool, Ship, and Standby Temperature.

Autosampler

Displays a graphical representation of the Autosampler tray, indicating the status of the active sequence (e.g., which pan positions have been configured in the active sequence and which runs are active or pending). In addition, this panel provides access to other Autosampler operations such as calibration functions and utilities (manually loading pans).

Method

Provides a summary of the sample and method information for the current run and allows the user to manually advance the method.

Utilities

Displays system health and shut down button.

Main Menu Options

System Info

  • The Network button displays the network configuration. Press Network to edit the configuration.

The Name and Location fields can be customized. Selecting one of these fields displays the keyboard. Once complete, press Save to accept the changes. 

  • The About button displays the versions for instrument and accessories.

Signals

Displays the real-time signal data generated directly from the instrument. Up to six signals can be displayed. Press Chart to view the signals in graph form.

To change the signal being displayed, press the signal you wish to change and then select the desired signal from the list:

Settings

Turn the Event and Air Cool on or off, prepare the Autosampler for shipping, and change the Standby Temperature.

Method

Provides a summary of the method information for the current run.

Autosampler

Displays a graphical representation of the Autosampler tray. Press a pan number on the tray image, and then press the R (Reference) or S (Sample) position to load the cell. To unload the cell, press the R or S (whichever is loaded) position.

Utilities

Displays the system health and diagnostics, and instrument Shutdown button.

Primary Function Buttons

Use the following keys for the main functions of the instrument.

 

Primary Function Buttons on the User Interface
Button Description

Start

Begins the experiment. This is the same function as Start on the instrument control software. Start automatically loads the selected sample pan and closes the furnace before beginning the experiment.

Stop

If an experiment is running, this button ends the method normally, as though it had run to completion; i.e., the method-end conditions go into effect and the data that has been generated is saved. This is the same function as Stop on the instrument control software. If an Autosampler sequence is in progress, Stop halts the sequence.

Lid Open/Close

Toggles between opening and closing the Lid on the DSC cell.

Reset

Re-homes all three axes and gripper and then shuts the lid, clearing from the instrument any information about the pans loaded in the cell.

Back

Returns to the previous screen.

Home

Return to the main menu.
