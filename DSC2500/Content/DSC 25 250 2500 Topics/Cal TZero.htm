Overview

The DSC Tzero* calibration requires two experiments. The first experiment is done without samples or pans (baseline); the second is performed with large (approximately 95 mg) sapphire disks (without pans) on both the sample and reference positions. Both experiments use the same method beginning with an equilibration at an initial temperature, followed by holding isothermal for 10 minutes, heating at constant rate to a final temperature and holding isothermal for 10 minutes. The range of temperatures should be at least as broad as the expected range of subsequent experiments. Tzero calibration should be done at relatively high heating rates such as 20°C/min in order to obtain the most accurate calibration of the sensor thermal capacitance and resistance values. Rates of less than 10°C/min are not recommended for Tzero calibration.

* Tzero™ describes proprietary technology invented by TA Instruments-Waters LLC and patented by TA Instruments-Waters LLC (U.S. Patent No. 6,431,747).

 

Recalibration is recommended if the type of purge gas or the cooling accessory has been changed. Recalibrating periodically is also recommended, especially if quantitative performance has degraded over time.

Follow these basic instructions for Tzero calibration:

  1. Collect Tzero calibration data, which requires two experiments—one run without samples or pans and one run with sapphire disks on the reference and sample positions (without pans).
  2. Analyze the data.
  3. Apply calibration results.

Collecting Tzero Calibration Data

  1. Access the File Manager and select the Calibration panel. Then click Calibration Setup. When you create a run from the Calibration Experiment Setup page, the settings are automatically saved.

  2. The Calibration Experiment Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.
  3. Select the Tzero checkbox to open the calibration form.

  1. Enter the following information.
  2. Enter the Sample Name .
  3. Enter Sample pan position for sapphire disk (applicable to second run), along with the sapphire mass and pan mass.
  4. Select the Reference drop-down menu to select the pan position for sapphire disk (applicable to second run). Set the sapphire mass. If the pan position and sapphire mass are not in the table, click Edit Tray Configuration at the bottom of the table to enter applicable values.
  5. Enter the Operator name.
  6. Enter the Project name, if desired.
  7. Enter any comments or Notes, if desired.
  8. Modify method parameters, if required.
  • Enter the Ramp rate, Lower temperature and Upper temperature limits, and Isothermallimit. The range of temperatures should be at least as broad as the desired experimental range of subsequent DSC experiments.
  • Tzero calibration should be done at relatively high heating rates such as 20°C/min in order to obtain the most accurate calibration of the sensor thermal capacitance and resistance values. Rates of less than 10°C/min are not recommended for Tzero calibration.
  • Select Perform Verification Run After Calibration to perform a baseline verification experiment after the Tzero calibration is applied. Note that there is no verification criteria associated with this experiment and that it is for manual inspection only.
  1. When complete, select from one of the following options:
  • Run All: Sends the two calibration experiments to the Running Queue and execute run sequence.
  • Queue All: Sends the two calibration experiments to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Manually Analyzing Tzero Calibration Data

Performing the calibration through the Calibration Setup page automatically analyzes the data and applies the calibration. Using the instructions below to analyze the data manually, if necessary, once the calibration experiments complete.

  1. Open the two calibration files in TRIOS.
  2. Select the two files in the File Manager Results pane. Alternatively, you can select this option from the File Manager right-click pop-up menu.
  3. When you select Calibrate > Tzero, a new document is created that contains the associated calibration results.

Several views are available within this document.

  • Spreadsheet: A table view of the associated calibration results (Capacitance and Resistance values)
  • Graph: A graphical view of the associated calibration results.
  • Results: A summary of the calibration parameters.
  1. Before saving the results, view the calibration graph. The profile is unique to each cell, but the graph should have the following general characteristics. If the profile doesn’t meet the criteria below, repeat the experiments. If problem persists, contact TA Instruments Service.
  • X-shaped profile with capacitance values decreasing and resistance values increasing
  • Smooth curve shape with no spikes

  1. On the Results view, click Apply to send the results to the instrument and enable the calibration.
  • NOTE: This calibration document will automatically be saved to the following folder: C:\ProgramData\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows 7 or 8.
