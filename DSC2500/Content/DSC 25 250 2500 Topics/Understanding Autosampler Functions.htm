There are several functions that the Autosampler can perform outside of the usual experiments. These functions help keep the Autosampler in good operating condition and help guarantee reproducible results.

Resetting the DSC Autosampler

Occasionally, you may find it necessary to reset the Autosampler. The reset function clears all commands, instructs the unit to locate the home positions, and returns the Autosampler to the park position. This function also instructs the Autosampler to clear the current sample and reference pan locations.

The Autosampler assumes that the cell is empty when you reset it, so ensure that this is true whenever resetting the unit. If there are any pans in the cell after resetting the Autosampler, remove them manually with tweezers. Otherwise, the Autosampler will not know that a pan is still in the cell and will attempt to load a second pan onto the first. This could cause damage to the pans. Always check that the cell is empty after the reset function is selected.

To reset the Autosampler, follow one of the following steps:

  • From TRIOS software, click Reset Autosampler available by accessing the Auto Sampler tab of the Control panel within TRIOS (screen shown below).

  • From the instrument user interface, press Reset .
