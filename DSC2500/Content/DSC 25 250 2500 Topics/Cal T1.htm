Overview

The baseline slope and offset calibration involves heating an empty cell through the entire temperature range expected in subsequent experiments. The calibration program is used to calculate the slope and offset values needed to flatten the baseline and zero the heat flow signal. This calibration is used in the DSC25, and when the selected heat flow is T1 for the DSC250 or DSC2500.

  • NOTE: MDSC experiments are available for DSC2500, DSC250, and DSC25.
  • NOTE: The Quench Cooler can only be used with Heat Flow T1. Change the heat flow selection or change the cooler type so that a compatible cooler is used with the heat flow you choose.
  • Baseline Slope is a calibration constant that accounts for the baseline slope of an empty cell. The baseline slope is calculated from data gathered as an empty cell is heated through the temperature range expected in subsequent experiments. Default is 0.00.

  • Baseline Offset is a calibration factor that shifts the curve up or down along the Y-axis to balance the heat flow at the sample and reference thermocouple positions. The baseline offset is calculated from data gathered as an empty cell is heated through the temperature range expected in subsequent experiments.

Collecting T1 Calibration Data

Baseline calibration involves heating an empty cell through the entire temperature range expected in subsequent experiments. The calibration program is used to calculate the slope and offset values needed to flatten the baseline and zero the heat flow signal.

  1. Access the File Manager and select the Calibration panel. Then click Calibration Setup.

  2. When you create a run from the Calibration Experiment Setup page, the settings are automatically saved.

  3. The Calibration Experiment Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.
  4. Select the T1 Baseline checkbox to open the calibration form.

  1. Enter the following information.
  2. Enter the Sample Name.
  3. Enter the Operator name.
  4. Enter the Project name, if desired.
  5. Modify method parameters, if required.
  • Enter the Ramp rate, Lower temperature and Upper temperature limits, and Isothermal limit. The range of temperatures should be at least as broad as the desired experimental range of subsequent DSC experiments.
  • Select Perform verification Run After Calibration to perform a baseline verification experiment after the T1 calibration is applied. Note that there is no verification criteria associated with this experiment and that it is for manual inspection only.
  1. When complete, select from one of the following options:
  • Run All: Sends the calibration experiment to the Running Queue and executes the sequence.
  • Queue All: Sends the calibration experiment to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Manually Analyzing T1 Calibration Data

Use the instructions below to manually analyze an existing T1 calibration file.

  1. Open the calibration file in TRIOS.
  2. Select the file in the File Manager Results pane.
  3. Right-click the file to display the File Manager context menu.
  4. When you select Calibrate > T1 Baseline, a new document is created that contains the associated calibration results.

Several views are available within this document.

  • Spreadsheet: A table view of the associated ramp data.
  • Graph: A graphical view of the associated ramp data.
  • Results: A summary of the calibration parameters.
  1. On the Results view, click Apply to send the results to the instrument and enable the calibration.
