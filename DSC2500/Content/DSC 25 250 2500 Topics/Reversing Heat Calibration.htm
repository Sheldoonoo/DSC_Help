Overview

Reversing Heat Capacity calibration is only required to measure the accurate value of the sample’s heat capacity or when using periods less than or equal to 40 seconds. The calibration uses a standard material with a known heat capacity over a specific temperature range of interest. Reference data is available for sapphire and polystyrene. Users can also generate their own reference tables for other materials. The calibration experiment should be run under similar conditions (pan type, modulation amplitude, period and heating rate) to those that will be used for subsequent MDSC samples. The range of temperatures should be at least as broad as the desired experimental range.

The Reversing Heat Capacity calibration curve is calculated by dividing the theoretical value of heat capacity by the measured value over a specific temperature range. The resultant calibration data, as a function of temperature, is sent to the instrument and applied to subsequent measurements.

This calibration only applies to the measurement of the Reversing Heat Capacity using T4P Heat Flow.

Follow these basic instructions for Reversing Heat Capacity calibration.

  1. Collect Reversing Heat Capacity calibration data, which requires heating a heat capacity calibration standard (in pans). The experimental method used will be similar to that for subsequent experiments. The range of temperatures should be at least as broad as the desired experimental range using the same conditions to be used in subsequent measurements (modulation conditions and purge gas).
  2. Analyze the data.
  3. Apply calibration results.

Collecting Reversing Heat Capacity Calibration Data

Reversing Heat Capacity calibration is set up using Calibration Setup from the Global Calibration page.

Sample Preparation

  1. Obtain the larger sapphire disk from the MDSC Calibration Sapphire Kit (P/N 970370.001) or other heat capacity calibration standard and record the sample weight.
  2. Weight an empty sample and reference pans and record their weights.
  3. Crimp the calibration standard into the sample pan, and then position it on the sample platform (or within an available Autosampler tray position).
  4. Prepare a reference pan using the same pan type used for the sample (empty pan). Crimp the pan, then position it on the reference platform (or within an available Autosampler tray position).

Reversing Heat Capacity Calibration

Before you begin, verify that the appropriate T1 Baseline, Tzero and Cell Constant/Temperature calibrations had been run previously on the instrument, they must have been performed, analyzed, and applied before beginning the Reversing Heat Capacity calibration procedure. These calibrations do not need to be performed with the new heat capacity calibration that is going to be run.

  1. Access the File Manager and select the Calibration panel. Then click Calibration Setup.
  2. When you create a run from the Calibration Setup page, the settings are automatically saved. The next time TRIOS is started, you will be asked if you would like to use your saved settings. Click Yes to populate the fields with your previously entered data, or No to receive a blank experiment setup page.

  3. The Calibration Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.
  4. Select the Reversing Heat Capacity checkbox to open the calibration form.

  1. Enter the following information.
  2. Enter the Sample Name.
  3. Enter the Sample pan position for the calibration sample along with the sample size and pan mass.
  4. Enter Reference pan number. Click the Reference field to display the drop-down table. Select the pan position along with the sample mass. If the pan position and sample mass are not in the table, click Edit Tray Configurationat the bottom of the table to enter applicable values.
  5. Select the standard from the Standard drop-down menu.
  6. Enter the Operator name.
  7. Enter the Project name, if desired.
  8. Enter any comments or Notes, if desired.
  9. Modify the method parameters.
  • Calibration must be performed using the same atmosphere and cooling accessory that will be used in subsequent experiments. Obviously, the lower and upper temperatures chosen for calibration must be within the operating range of the cooling accessory selected. In addition, the temperature range for calibration should be at least as broad as the range that will be used in subsequent experiments. Typical temperature ranges are:–120 to 400°C for the RCS120, –90 to 550°C for the RCS90, –40 to 400°C for the RCS40, and 50 to 400°C for the FACS.
  • Enter the Modulation parameters and the heating Ramp rate. Heat the material at the same heating rate that you will use in your subsequent experiments.
  • Enter the Start Temperature and Final Temperature.
  1. When sample and experimental input is complete, select from one of the following options:
  • Run All: Sends the calibration experiment to the Running Queue and execute run sequence.
  • Queue All: Sends the calibration experiment to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Manually Analyzing Reversing Heat Capacity Calibration Data

Once the calibration experiment completes, analyze the data following the instructions below.

  1. Open the Reversing Heat Capacity calibration file in TRIOS.
  2. Select Calibrate > Heat Capacity in the File Manager Results pane. Alternatively, you can select this option from the File Manager right-click pop-up menu.
  3. When you select Calibrate > Heat Capacity, a new document is created that contains the associated calibration results.

Several views are available within this document.  

  • Spreadsheet (the first Calibration tab in the above image): A table view of the associated calibration results.
  • Graph (the second Calibration tab in the above image): A graphical view of the associated calibration results.
  • Results Summary (the Calibration Results tab in the above image): A summary of the calibration parameters.
  1. On the Results view, click Apply to send the results to the instrument and enable the calibration.

  • NOTE: This calibration document will automatically be saved to the following folder: C:\ProgramData\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows 7 or 8.

Selecting Reversing Heat Capacity Calibration Results

The Discovery DSC stores all the calibration results in a database internal to the instrument. This allows you to access and set calibrations accordingly to meet the needs of your experiment. Selection of the calibration set is made on the Calibration Data page.

After analyzing and saving the results, you can apply the results so they are used in future experiments.

Access the Calibration > Calibration Data window from the File Manager to access the Calibration Data page.

Verify the desired calibration set is selected after applying the new calibration result.
