Overview

The specific heat of an unknown material can be determined using any of the three heat flow types (T1, T4, or T4P) by comparing the difference between the sample’s heat flow signal and the heat flow signal of a known heat capacity standard (typically sapphire) relative to a common heat flow baseline. This is considered the classic 3-run heat capacity method discussed in ASTM E1269.

Collecting Classic 3 Run Heat Capacity Data

To obtain the best results from the DSC Heat Capacity analysis, the data used should follow these guidelines:

  • The sample pans used to obtain the data files should be matched in material and type.
  • Pan/lid combinations for all runs should be cleaned and weight matched to a precision of + 0.01 mg.
  • Samples and reference materials should be weighed to a precision of + 0.01 mg.
  • Samples should be distributed uniformly across the bottom of the pan. The pan should be crimped so that the bottom is as flat as possible.
  • Volatile samples should be sealed in hermetic pans and weighed after sealing.

  • NOTE: Encapsulate the sample as tightly as possible to minimize temperature gradients and/or sample movement.
  • Small sample sizes should be used to reduce thermal gradients. ASTM (E1269) recommends 5–15 mg sample sizes for organics and polymers, and 20–50 mg sizes for inorganics. In addition, sample mass is assumed to be constant in the heat capacity equation. The sample pan should be weighed before, as well as after, the experiment to ensure that no weight loss has occurred due to vaporization or decomposition. If a weight change > 0.3 percent is observed, the measurement is invalid.
  • Three data files will be needed:
    1. Sample File—containing data from an experiment using an unknown sample.
    2. Baseline File—containing data from an experiment using empty DSC pans. When setting up the baseline experiment in TRIOS, the sample size must be set to zero in order to enable the analysis.
    3. Reference File—containing data from a heat flow calibration run using a known, well- characterized reference sample (sapphire). When setting up the reference experiment in TRIOS, the sample name must be sapphire in order to enable the analysis.
  • All three experiments should be run using the same Iso-ramp-Iso procedure.
  • Include a ten-minute isothermal period both before and after the ramp segment to establish equilibrium points between the three experiments.
  • Temperature calibration of the DSC is strongly recommended. This calibration should be made using high purity (99.99%) calibration materials such as indium or other suitable material.
  • A ramp rate of 20°C/min is recommended because it provides the best compromise between maximum heat flow displacement and minimum thermal gradients.

Performing Classic 3 Run Heat Capacity Analysis

Once the calibration experiment completes, analyze the data following the instructions below.

  1. Open your baseline, reference, and sample data files in TRIOS.
  2. Select all three files in the File Manager Results pane.
  3. Right-click and select Transformations… > Heat Capacity Calculation…

  1. Verify the information displayed in the dialog, then click OK.

An overlay document is generated with ASTM Heat Capacity and Heat Flow plotted versus Temperature. In addition to this overlay document, the new ASTM Heat Capacity signal is made available in your sample’s data file.
