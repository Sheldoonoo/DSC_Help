The Table View provides a quick summary of all the experimental parameters for the sequence. Additional runs can be created in the Table View by simply appending runs down from a selected row/experiment, thereby duplicating all the experimental parameters. Specific experimental conditions can be readily modified, copied and pasted between fields, or filled down or up within a given column. In the Table View, multiple runs can be quickly created and easily verified as appropriate experimental conditions by the operator. Editing Table Parameters

To edit a value in a Table View cell, click within that cell. Depending on the parameter, enter or select a new value. Clicking away from that field or pressing Enter on the keyboard saves the new information.

Copying Parameters

The Table View allows for easily copying parameters from run to run using fill down/up functions.

To copy parameters:

  1. Select the desired parameter (cell) to copy. A green box appears in the lower right corner of the cell.

  1. Left-click the green box with the mouse and hold down while dragging the box down, selecting the cells in that column as desired.

  1. Release the mouse when satisfied with the cell range. {This same concept can also be applied with more than one column selected.}

Alternatively, you can also select this option using the right-click pop-up menu.

 Customizing the Table View 

Customizing the Table View is important, as there is a limited viewing area for displaying this information. The parameters (columns) should be customized so that the operator has easy access to the run information that is likely to require modification or verification. In addition to column selection, the column order, column width, and color settings can also be customized.

 

Reordering Columns 

There are two methods to reorder the columns within the Table View: using the Column Selector window or dragging/dropping the column as described below.

Click within the column you wish to move, and hold down the mouse while moving the column to the desired column location. Arrow indicators appear, showing where the column will be upon releasing the mouse.

 

Locking Columns in Place

If the Table View is configured such that a scroll bar is needed to view all the information, you may find it useful to lock some columns in place. These locked columns are always visible while you scroll.

To lock a column, click on the pin symbol on the column header. To unlock, click again. Any locked columns relocate to the top of the column order.

 

Defining Table Properties

Individual components within TRIOS are customized through the Properties panel. To access this panel:

  1. With the desired Table View displayed, click View tab > Properties.
  2. The associated settings for the selected items display.
