Discovery DSC Cell Calibrations

To obtain the most accurate experimental results, calibrate the DSC cell upon initial installation and recalibrate periodically. These experiments are performed using TRIOS software. For details on how to perform calibrations, refer to appropriate linked topics in this section.

Calibration consists of several different types of procedures, which are described briefly below.

  • NOTE: Perform calibration runs that encompass the temperature range you plan to use in your experiments. (We recommend calibration over the widest temperature range.) If your subsequent experiments cover a wider range, you may wish to recalibrate.
  • NOTE: Please make sure that you run your experiments with the same purge gas that you used to calibrate the system. For example, if you calibrate using a nitrogen purge, perform your subsequent experiments with a nitrogen purge.

Tzero® Calibration*

The DSC Tzero calibration requires two experiments. The first experiment is done without samples or pans (baseline); the second is performed with large (approximately 95 mg) sapphire disks (without pans) on both the sample and reference positions. Both experiments use the same method, beginning with an equilibration at an initial temperature, followed by holding isothermal for ten minutes, heating at constant rate to a final temperature, and holding isothermal for ten minutes. The temperature range should be at least as broad as the expected range of subsequent DSC experiments. Tzero calibration should be done at relatively high heating rates (such as 20°C/min) in order to obtain the most accurate calibration of the sensor thermal capacitance and resistance values. Rates of less than 10°C/min are not recommended for Tzero calibration.

* Tzero® describes proprietary technology invented by TA Instruments—Waters LLC and patented by TA Instruments—Waters LLC (U.S. Patent No. 6,431,747).

 

T1 Baseline Calibration (Discovery DSC2500, DSC250, and DSC25 Only)

The baseline slope and offset calibration involves heating an empty cell through the entire temperature range expected in subsequent experiments. The calibration program is used to calculate the slope and offset values needed to flatten the baseline and zero the heat flow signal. Baseline Slope is a calibration constant that accounts for the baseline slope of an empty cell. The baseline slope is calculated from data gathered as an empty cell is heated through the temperature range expected in subsequent experiments. Default is 0.00. Baseline Offset is a calibration factor that shifts the curve up or down along the Y-axis to balance the heat flow at the sample and reference thermocouple positions. The baseline offset is calculated from data gathered as an empty cell is heated through the temperature range expected in subsequent experiments.

 

Enthalpy (Cell) Constant

The Enthalpy (Cell) Constant calibration is based on an experiment in which a calibration melting standard (for example, indium) is heated through its melting transition. The calculated heat of fusion is compared to the theoretical value. The cell constant is the ratio between these two values. The onset slope, or thermal resistance, is a measure of the suppression of temperature rise that occurs in a melting sample in relation to the measuring thermocouple. Theoretically, a standard sample should melt at a constant temperature. As it melts and draws more heat, a temperature difference develops between the sample and the sample thermocouple. The thermal resistance between these two points is calculated as the onset slope of the heat flow versus temperature curve on the front of the melting peak. The onset value is used for kinetic and purity calculations to correct for this thermal resistance.

The prerequisite for Enthalpy Cell Constant calibration is Tzero calibration.

 

Temperature

Temperature calibration is based on an experiment in which a temperature standard (for example, indium) is heated through its melting transition. The extrapolated onset of the recorded melting point of this standard is compared to the known melting point, and the difference is recorded for temperature calibration. The same file used for the cell constant calibration can be used for this calibration.

In addition, you can use up to four other standards to calibrate temperature. If you use three or more standards, the temperature correction is modeled by a cubic spline fit. The multiple-point temperature calibration is more accurate than the one-point calibration if absolute temperature measurements are required over a broad (>300°C) temperature range. If multipoint temperature calibration is used, it is recommended that you choose standards that will bracket the desired measurement temperature range. This will ensure optimal calibration and performance.

The prerequisite for Temperature calibration is Tzero calibration.

  • Use caution when using reference materials other than Indium, as several of these materials alloy with aluminum pans or have less certain known values for the onset temperature.

Standard Heat Capacity

Standard Heat Capacity calibration can be obtained in a single experiment with the Discovery DSC. Heat capacity calibration is required to measure an accurate value of the sample’s heat capacity. The calibration uses a standard material with a known heat capacity over a specific temperature range of interest. Reference data is available for sapphire and polystyrene. You can also generate your own reference tables for other materials. Calibration should be done at a heating or cooling rate of 10 to 20°C/min.

The heat capacity calibration curve is calculated by dividing the theoretical value of heat capacity by the measured value over a specific temperature range. The resultant calibration data, as a function of temperature, is programmed in the instrument.

The prerequisites for Standard Heat Capacity calibration are Tzero, Cell Constant, and Temperature calibrations.

 

MDSC Heat Capacity

MDSC Heat Capacity calibration is only required to measure the accurate value of the sample’s heat capacity or when using periods less than or equal to 40 seconds. The calibration uses a standard sapphire material with a known heat capacity over a specific temperature range of interest. Reference data is available for sapphire and polystyrene. You can also generate your own reference tables for other materials. The calibration experiment should be run under similar conditions (pan type, modulation amplitude, period, and heating rate) to those that will be used for subsequent MDSC samples. The heating rate can be set to the same as subsequent experiments or a nominal value of 5°C/min can be used.

The Reversing Heat Capacity calibration curve is calculated by dividing the theoretical value of heat capacity, as a function of temperature, by the measured value. The resultant calibration data, as a function of temperature, is programmed in the instrument.

The prerequisites for MDSC Heat Capacity calibration are Tzero, Cell Constant, and Temperature calibrations.

When to Calibrate

  • For optimum experimental results, perform all DSC cell calibrations whenever you change one of the following parameters:
  • First use of a new cell
  • Installation of new lid
  • Purge gas change
  • Cooling device or accessory (FACS, RCS, or Quench Cooler) change
  • Perform the additional Discovery DSC calibrations in intervals specified in the table.

Basic Calibration Steps

Some calibrations are dependent upon other calibration results; therefore, calibration is performed in a hierarchical order as described below.

  1. Tzero Calibration
  2. Set up and run Tzero calibration experiments.
  3. Analyze and apply Tzero calibration results.
  4. T1 Calibration (Discovery DSC2500, DSC250, and DSC25 only)
  5. Set up and run T1 calibration experiments.
  6. Analyze and apply T1 calibration results.
  1. Cell Constant and Temperature
  2. Set up and run Cell Constant and Temperature calibration experiments.
  3. Analyze and apply Cell Constant and Temperature calibration results.
  4. Heat Capacity (optional)
  5. Set up and run Standard Heat Capacity calibration experiment (if desired).
  6. Set up and run MDSC Heat Capacity calibration experiment (if desired).
  7. Analyze and apply standard heat capacity calibration results.

Auto Calibration and Scheduling Runs

You can schedule your calibrations to run automatically by accessing Calibration Setup. Calibrations can be run immediately, sent to a queue, or scheduled to be run at a later date.

TRIOS must be connected to the instrument, open, and running, for the scheduled experiments to run. If TRIOS is opened after the time a run was scheduled for, then the next experiment or sequence in the schedule list will run at its scheduled time.

Setting Up Auto Calibration

  1. Select the Calibration tab from the File Manager.
  2. Click Calibration Setup.


  3. The Calibration Setup tab opens. Select the calibration type(s) you wish to perform by selecting the applicable boxes. Additionally, select Cell Conditioning and/or Baseline Conditioning , if required. For more details on creating an auto calibration or schedule for a specific calibration, refer to the links below.
    • Tzero
    • Cell Constant/Temperature – both Cell Constant and Temperature calibrations are performed. Select Temperature to run a Temperature-only calibration.
    • Temperature
    • Heat Capacity
    • Reversing Heat Capacity

    Each calibration opens in a new tab, in the order in which the calibration should be performed.


  1. Set your calibration experiment preferences in each calibration tab.
    • Click Add Experiment to add as many experiments as needed (only applicable to Temperature calibration and verification experiments). Use the drop-down menu on the Reference to change the reference type. Once an experiment is added, the Remove Experiment button becomes available.

    • Select Calibration to perform a calibration experiment. Select Perform verification after calibration to run a verification experiment after the calibration is generated and applied to the instrument.
    • Select Verification to perform a verification experiment. Select Perform calibration if verification fails to run a calibration experiment if the verification run fails to meet set criteria. If the calibration fails, a warning message displays and the sequence will continue with the next item in the Running Queue.
  2. Verify the file path. Specify a new file path if the calibration files are to be saved in a location other than the default location.
  3. Select Run All, Queue All, or Schedule All.
    • Run All–Sends the calibration experiment to the Running Queue and executes the sequence.
    • Queue All–Sends the calibrations to the Running Queue, where they can be viewed before running. If there are already runs listed in the queue, the added calibrations are appended to the bottom of the list. Calibrations in the queue can be run by clicking Start on the Ribbon or will automatically run in the order in which they are listed in the queue when the instrument is in an active state.
    • Schedule All– Schedules the calibration to run at a later specified date. The Add New Schedule dialog box allows you to select how often the calibration should be run, the time, and the begin and end date.

Scheduling

Once you have added a new schedule (by clicking Schedule All in Calibration Experiment Setup), access the Experiments tab in the File Manager > Schedule tab to view and make changes to scheduled runs. A log of the scheduled run actions appears at the bottom of the page.

 

  • NOTE: For the calibration to run, the Enabled box must be checked, the instrument active, and TRIOS running .

Editing a Scheduled Run

  • Use the drop-down boxes to change the Frequency (Once, Daily, Weekly, or Monthly), Time, Begin Date, End Date.
  • To delete a scheduled run, highlight the row by clicking in one of the cells. The Delete button becomes enabled.

Calibration Summary

A summary of the current calibration can be accessed on the Calibration Data window in TRIOS software.
