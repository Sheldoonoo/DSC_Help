This function is used to define the default settings for each DSC calibration standard, add new standards, and reset (reload) the default calibration table. These settings include: the theoretical values use for calibration for each standard.

Accessing Calibration Standards

To define calibration standards:

  1. Click the TRIOS button , located on the top left corner of the application. The window below displays.

  1. Click Options (circled in the image above). For additional information related to the Options window, see Setting TRIOS Options.
  2. For Temperature/Cell Constant standards, click Discovery DSC > Temperature Cal. The window that displays allows you to customize the existing calibration standards and also add new ones.
  1. For Heat Capacity standards, click Discovery DSC > Heat Capacity. The window that displays allows you to add new Heat Capacity standards.

Editing Calibration Standards

To edit the theoretical values in the DSC Temperature Cal Standards list:

  1. Access the DSC Calibrations Standards options window.
  2. Click within the desired field and enter the new value.
  3. Click OK to save the changes. Any entries made here are saved for future recall.
  4. To revert the DSC Standards parameters list back to the default values, click Reset.

To edit the theoretical values in the DSC Heat Capacity Standards list:

  1. Access the DSC Heat Capacity Standards options window
  2. The Sapphire and Polystyrene standards are not editable, but User standards can be edited as follows:
  • Click within the desired fields and either enter a new value or edit a current value.
  • Click Paste to paste in values from excel.
  • Click Load to load a .csv file into the table.
  1. Click OK to save the changes. Any entries made here are saved for future recall.

Adding Calibration Standards

To add other standards to the DSC Temperature Cal Standards list:

  1. Access the DSC Calibrations Standards options window.
  2. Click Add Standard and enter the requested information into the new entry within the table.
  1. Enter a Standard Name. This standard and its settings are saved when you click OK.
  2. Enter the literature Melting Temp (in Â°C). Consult your Chemistry handbook for the proper value.
  3. Enter the literature Enthalpy in Joules/gram. Consult your Chemistry handbook for the proper value.
  4. Click OK to save the changes. Any entries that are made here are saved for future recall.
  1. NOTE: To return all of the standards in the Standards List back to the default values, click Reset All.
  2. NOTE: The Delete button can be applied only to a standard that you have entered. You cannot delete the default (factory) TRIOS software standards.

To add other standards to the DSC Heat Capacity Standards list:

  1. Access the DSC Heat Capacity Standards options window.
  2. Click Add Standard to add a standard.
  3. Edit the name by modifying the Standard Name Field.
  4. Edit the Lower Temperature and Upper Temperature values to set the start and end temperature range used in the template when setting up the experiment.
  5. NOTE: To remove the standard from the list, click Remove.

In the event that you want to restore the entire calibration table back to the original manufacture defaults (set at TA Instruments), click Reset All.
