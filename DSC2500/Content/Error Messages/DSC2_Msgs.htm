6064 LN2P Tank Empty–Liquid level sensor reports 0% full You have set up a run that requires the LN2P to provide liquid nitrogen cooling, but the LN2P is empty. The run will not be started until the LN2P contains adequate liquid nitrogen. If this error occurs during a run, the run will be terminated, the LN2P turned off, and the cell will go to standby temperature. Fill the LN2P tank using one of the methods outlined in the LN2P Getting Started Guide or the Discovery DSC Getting Started Guide.
6065 Liquid Level Sensor Out of Range {0:float} Liquid Level Sensor Out of Range. Low = possible short circuit of the sensor. High = possible open circuit of the sensor. Contact TA Instruments for service.
6066 LN2P Tank Level Not Increasing During Fill: {0:float} Bulk tank may be empty. The LN2P liquid level sensor is not changing during the autofill function, indicating that the bulk storage tank may have run out of liquid nitrogen.

Check the condition of the bulk storage tank and resume autofill.

If this error persists, contact TA Instruments for service.
6067 LN2P Tank Level Warning–Level of LN2 remaining in the tank reaches 20% of full. The LN2P liquid level sensor indicates that your tank is only 20% full of LN2 You may want to refill your tank now or use the autofill feature prior to your next run to ensure adequate cooling supply to the cell for your experiment. If this message occurs during a run, the run will continue.
6082 Auto Sampler Crc failed. Please reload firmware. The Autosampler board firmware failed a confidence check and may be corrupt.

Reload Autosampler firmware to the instrument from the PC and cycle power to the instrument.

If this error persists, select "Bypass Autosampler" to run the instrument in manual mode and contact TA Instruments for service.
6083 Auto Sampler failed power on self test (POST) The Autosampler failed its power-on self-test. If this error persists, select "Bypass Autosampler" to run the instrument in manual mode and contact TA Instruments for service.
6084 Auto Sampler Autocal Failed Verification The DSC Autosampler failed to confirm its calibration.

Inspect the DSC cell and autosampler tray to verify that each is installed properly and there are no obstructions. Select "Calibrate Autosampler" in instrument control to attempt the calibration and verification again.

If this error persists, select "Bypass Autosampler" to run the instrument in manual mode and contact TA Instruments for service.
6085 Autosampler Tray Not Present–tray limit switch did not detect tray present at method start or during method start or during load/unload operation. The Autosampler tray limit switch did not detect the tray present at method start or during a load/unload operation. The switch may have failed or you may have neglected to place the sample tray on the instrument before attempting to run your method.

If the tray is missing, replace the tray on the instrument and retry.

If this error persists, select "Bypass Autosampler" to run the instrument in manual mode and contact TA Instruments for service.
6086 Pan Not Detected – The range finder did not detect a pan when a pan was expected in the gripper fingers. The Autosampler failed to detect a pan during the load/unload operation. Visually inspect the DSC cell or Autosampler tray for a missing pan. Replace the missing pan and resume operation. If the pan is present but not detected, there could be a hardware problem with the Autosampler. Retry the operation and if the problem persists, contact TA Instruments for service.
6087 Auto Sampler Encountered Unexpected Object – Bounce sensor triggered. Halt motion and travel full up and stop.

The Autosampler detected an unexpected object during y-axis motion and was unable to recover.

This could be caused by:

  • An obstruction in the axis of motion
  • A failed motor or lead screw
  • An Autosampler wire harness problem
  • An Autosampler board failure
 Reset the Autosampler in instrument control and retry operation.

  • An Autosampler wire harness problem
  • An Autosampler board failure

6091 Finger Motion Error

The Autosampler encountered a gripper finger motion error – the gripper finger position could not reach the desired position.

This could be caused by:

  • An obstruction in the gripper finger motion
  • A decomposed sample adhered to the gripper fingers causing the pan to become stuck to the fingers
  • A mechanical problem with the gripper finger rotation
  • A failed gripper motor
  • A failed absolute encoder
  • An autosampler wire harness problem
  • An autosampler board failure

Reset the Autosampler in instrument control and retry operation.

If this error persists, select “Bypass Autosampler” to allow you to run your instrument in manual mode and contact TA Instruments for service.
6092 Auto Sampler Dropped Pan (Near Cell Area) – After a successful pick-up but before a gripper open event, the range finder did not detect a pan when a pan was expected in the gripper fingers. The pan may have been dropped during travel from tray to cell or back. The Autosampler failed to successfully load/unload pan after pick-up. The pan may have dropped during travel in an area near or in the DSC cell proper.

Visually inspect the cell area for a dropped pan and resume operation (the Autosampler will resume operation with the next scheduled pan to be loaded).

If this error persists, select “Bypass Autosampler” to allow you to run your instrument in manual mode and contact TA Instruments for service.
6093 Run in Progress, Cannot Open Lid – Open lid selected from touchscreen or PC during an active method. You requested the lid to open during an active run. The lid should not be opened during an active experiment. Wait until the current run is complete before opening the cell. The lid will open even if running an experiment.
