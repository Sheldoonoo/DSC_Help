Understanding Error Messages in TRIOS

 

The Notifications Panel in TRIOS displays a current list of messages, along with the relative code (numeric value), date and time, and other pertinent information. See Using the Notifications Panel for more information on understanding and using this panel.

Use the code specified in the Notifications Panel and the table below to determine the category of the message being displayed. Click the applicable message code range in the table below to view a listing of the messages, as well as possible solutions to error messages.

 

Message Code Range Message Category
