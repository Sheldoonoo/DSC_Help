About the Guardian Audit Trail

There are various events that are recorded within the Notification Log when the Guardian option is enabled. These events include General, File Operations, and Instrument Control. Listed below are the events that are currently being logged. See Using the Notifications Manager for additional information regarding using this panel.

 

General Events (at TRIOS Launch)

  • Windows User ID and User Level
  • Computer Name
  • Version Information
  • Firmware Upgrades
  • Keys (Options) Enabled
  • Service Logins
  • Connections to Instruments (Machine ID)

File Operations Events

  • Procedures File Open (path and filename)
  • Procedure File Save (path and filename)
  • Sequence File Open (path and filename )
  • Sequence File Save (path and filename)
  • Method File Open (path and filename )
  • Method File Save (path and filename)
  • Design View Open (path and filename )
  • Design View Save (path and filename)
  • Data File Open (path and filename)
  • Data File Save (path and filename)
  • Notifications Log Open (path and filename)
  • Notifications Log Save (path and filename)
  • File Export (path and filename)
  • Print to PDF

Guardian Option Events

  • Enabling/Disabling of the Guardian Option
  • User Accounts (Adding, Removing, and Changing)
  • Instrument Control Events
  • Experimental Events that include starting and stopping experiments
  • Modifications to the running experiment
  • Instrument Events such as pan load/unloads
  • Quick Runs from Touch screen (Discovery Thermal)
  • Calibration changes

Other events, such as the details of the experimental conditions and results log (if enabled), are stored within the data file itself. This information can be viewed within TRIOS by right-clicking on the file in the Results Manager and then selecting Show Parameters and/or Show Results Log.

Data File Events

  • Version of TRIOS used to run experiment
  • Versions of embedded firmware used to run experiment
  • Windows User ID and Computer Name
  • Experimental Information
  • Instrument Options
  • Calibration Parameters
  • Original Data
  • Instrument events that occurred during experiment
  • Instrument message specific to experiment
  • Changes to original procedure
  • Analysis Log
  • Modifications to original data file
  • Sample Information
  • Geometry (applicable to Rheology)
  • Spreadsheet data
