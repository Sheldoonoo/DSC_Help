The appearance of many of the components in TRIOS can be customized. You can customize settings that apply to the entire program, or select properties that apply to individual components within the application.

  1. NOTE: The TRIOS software Application options are global settings applicable to the computer where the program is installed. Therefore, if you open more than one copy of the program, these settings are shared.

For an overview of document properties and how they work, refer to Understanding Document Properties.

Customizing the Application Settings

To select the application options, follow these steps:

  • Click the TRIOS button at the top left corner of the application. This opens the window shown below.

  • Click Options (circled in the image above). This display window allows you to customize user settings, display settings, and file settings. For more information on Options, refer to the Help topic Setting TRIOS Options.
  • Choose the desired settings and click OK.

Accessing and Customizing Properties

Individual components within TRIOS are customized through the Properties panel.

To access this panel:

  • With a TRIOS document open, click on the desired component (such as a graph, for example).
  • Right-click and select Properties from the pop-up menu. The associated settings for the selected items display.
  • NOTE: You can display the Properties panel for most components by double-clicking on that component. For example, double-click on an axis to open the Properties panel for that axis.

Follow the tips below when using a Properties panel:

  • Choose a component for customization by displaying the Properties panel, then selecting the desired component from the drop-down list at the top of the panel. [This list will contain only those items available within the same component.]
  • Save and restore the selected properties by using the icons at the top of the Properties panel.

To Customize the Spreadsheet Properties

General Spreadsheet

The spreadsheet properties allow you to customize the general properties (such as font and color) associated with the spreadsheet. Follow these steps to customize the general spreadsheet properties:

  • Click on the desired Spreadsheet view tab at the bottom of the window.
  • Right-click and select Properties from the pop-up menu.
  • NOTE: You can also double-click on the row header to open the Properties panel.
  • Select Spreadsheet Properties from the drop-down list.
  • Click on the desired property to display either an arrow or button . Then choose your option from the drop-down list or window.
  •  
    • All Cells: Choose the desired font and size, text color, and cell background color to apply to all cells in the spreadsheet.
    • Selection: You must first select the desired cells to change, then set the text color and background color using this property.
    • Alternating Row: If you want a different color for alternating rows in your spreadsheet, set "Enable" to Yes then select the desired color. If you do not want a different color for alternating rows (all cells will be the same), set "Enable" to No.
    • Headers: Use this property set to choose the desired color and font to be used for the column header text.
    • Miscellaneous : Set "Show blank space" to No to hide any empty cells or to Yes to show any empty cells. The color selected for "Modified flash color" is used as an indicator that data was modified.

    Spreadsheet Contents

    The spreadsheet content properties allow you to customize the selected variable, units, label, numerical format, and precision. (These settings are also available using the spreadsheet menu.) Follow these steps to customize the spreadsheet content properties:

    • Click on the desired Spreadsheet view tab at the bottom of the window.
    • Click on a column header.
    • Right-click and select Properties from the pop-up menu.
    • NOTE: You can also double-click on the column header to open the Properties panel.
    • Click on the desired property to display either an arrow or button . Then choose your option from the drop-down list or window.
      • Number format:  Select from the available formats—decimal, significant figures, scientific (notation), general, time, or angle. (The angle format only applies to specific variables.)
      • Precision: Select the desired value from one to ten. This defines the number of digits displayed, based on the number format selected.

    Axis – X1, Y1, Y2, Y3, and Y4

    The axis properties allow you to customize the axis settings such as variable selection, scaling, tick mark, and color preferences. Follow these steps to customize the axis properties:

    • Click on the desired axis.
    • Right-click and select Properties from the pop-up menu.
    • NOTE: You can also double-click on the axis to open the Properties panel.
    • Select the axis that you want to modify from the drop-down list.
    • Click on the desired property to display either an arrow or button . Then choose your option from the drop-down list or window.
    • Grid: To apply grid lines to a single axis’ tick divisions, select Major grid or Minor grid. You can also configure a single grid line through zero.
    • Grid (all axes): Select the grid color, line style, and line width from the list of available options.
    • Tick marks: Select from the displayed options to customize the tick mark display, labels, font type and size for major and minor ticks, and to customize log tick marks. Using these options you can turn the tick marks on or off. Select the position of the tick marks to display them on the outside, inside, or through the axis line. You can also choose to display ticks at major tick intervals, or display them at the minimum and maximum values of the plot.
    • Axis Line: Set "Axis line at value" to Yes to display the axis line at your entered value.
