Creating a New User Variable

  1. Open the file and display the Spreadsheet view by clicking the spreadsheet tab at the bottom of the viewing pane.
  2. Click the User Defined button from the Variables toolbar of the Format tab.
  3. The User Variables dialog box displays. Click New.

  1. Enter the Name of the new user variable.
  2. Optional: Enter the Equation Name.
  3. Optional: Enter the Symbol associated with the Name, which will appear with the Name on the axis of your graph and in the column of your spreadsheet) â€“ for example the symbol t for Time.
  4. Optional: Add Notes, if desired.
  5. Select the Format of the equation: Numeric or Text (Text allows you to use both numeric and alphanumeric characters).
  6. Select the Format of the data that appears in the spreadsheet (most commonly used are Numeric and Significant figures).
  7. Optional: Enter the Unit (for example, enter min as the unit for Time).
  8. To create your equation, make sure the Is calculated check box is selected. De-select this option to create an empty placeholder variable in which to create your own data.
  9. Create your equation in the Equation text field. To use any of the variables, functions, or constants provided in the lists, double-click the items in the list to add it to the equation.
  10. Click OK.

Adding a New User Variable to the Spreadsheet

Once the variable has been created, you can apply it to your graph as follows:

  1. Click Select from the Variables toolbar of the Format tab (Spreadsheet view).
  2. The Spreadsheet Columns dialog box displays.

  1. Select the Show user variables checkbox. The new user variable displays at the bottom of the Available variables list.
  2. Add the new user variable to the spreadsheet columns by clicking the right arrow. The variable will move to the bottom of the Displayed columns list. If desired, reorder the columns by selecting a variable and using the up and down arrows to change its position in the list, or remove variables from the list by using the left arrow.
  3. Click OK.

Adding a New User Variable to the Graph

  1. Right-click the graph and click Select variables.

  1. The Graph Variables pane opens. Select the axis from the Axis list, and then assign a variable by selecting the variable(s) from the Variables list. Click OK. Refer to Configuring Graph Variables for more information.
