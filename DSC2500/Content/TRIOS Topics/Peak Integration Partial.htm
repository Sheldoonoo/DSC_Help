Peak Integration – Divide Peak

Follow these basic steps to perform an Peak Integration analysis including divided areas:

  1. Click on the desired curve to analyze.
  2. Select Peak Integration from the Analysis > Function menu.
  3. Click Analyze.
  4. Click Edit Analysis.

  1. Check the Divide peak option to divide the area under the curve based on the Type selection:
  1. Equal x regions: Divides the main peak equally into the number of regions specified.
  1. Fixed x interval: Divides the main peak based on the x-interval specified.
  2. Specific x: Used to define the exact x-axis limits to divide the main peak. More than one entry can be specified by separating multiple entries with # (for example, 125#150).
  3. At peak: Divide the main peak into two areas at the peak position. Note that there is a maximum of 10 partial areas.

  1. Once the perpendicular (vertical) dividing lines are in place, you can adjust their position by clicking Adjust Drops, then manually repositioning the vertical dividing lines. Then click Finish Adjust.

  2. Click Accept to execute the analysis. Click Cancel to cancel the analysis functions.

Refer to Available Analysis Functions for details on Peak Integration options.
