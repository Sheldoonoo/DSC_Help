Available Analysis Functions

This topic covers the basic analysis functions in TRIOS software. The list of available functions varies based on the type of data obtained. For additional information, refer to Basic Steps to Analyzing Data Help topics.

Performing an Analysis

  1. Click on the desired curve to analyze.
  2. Select the desired analysis from the Analysis > Function menu.
  3. Click Analyzeto execute the analysis.

Use the Signal change option to measured the change in the measured values between two defined points of data. Glass Transition

  • Click here to watch a TA Tech Tip on analyzing Glass Transitions and Melt Peaks in TRIOS. Note that this video was recorded using a Discovery DSC.

Use the Glass transition analysis to calculate the onset, end, inflection, and signal change of a glass transition in the curve. For example, this analysis enables you to determine glass transition temperatures by DSC.

  • Onset is calculated as the intersection of the first and second tangents.
  • Inflection is the point on the curve between the first and third tangents with the steepest slope.
  • End is calculated as the intersection of the second and third tangents.
  • NOTE: The Glass transition option is not available if the Y-axis scale is logarithmic. See Basic Steps to Analyzing Data to select associated analysis options before beginning.
  1. NOTE: Peak Max and Signal Max analyses report the same result if the baseline is perfectly horizontal (the Y-axis values of the start and stop limits are the same).

Glass Transition Analysis Options

Onset type: Select between the following options:

  • Tangent: Uses the selected point to calculate a tangent line
  • Manual tangent: Allows you to define the start and stop limits for the onset tangent line

Transition type: Select between the following options:

  • Tangent: Uses the selected point to calculate a tangent line
  • Manual tangent: Allows you to define the start and stop limits for the inflection tangent line
  • Inflection: Determines the inflection point between the onset and end limits

End type: Select between the following options:

  • Tangent: Uses the selected point to calculate a tangent line
  • Manual tangent: Allows you to define the start and stop limits for the onset tangent line

Midpoint type: Select one of the following methods to be used in determining the midpoint in step transition analyses:

  • None: Results in no step transition midpoint calculation
  • Inflection: Results in the use of the inflection point (the point of data with the steepest slope) as the midpoint of the step transition
  • Half height: Defines the midpoint as the Y-axis value halfway between the calculated onset and calculated end of the step/glass transition region
  • Half width: Defines the midpoint as the X-axis value halfway between the calculated onset and calculated end of the step transition region
  • Half extrapolated tangents: Results in a line drawn midway between the calculated onset and end tangents, and the intersection of this line with the data is considered the midpoint
  • Fictive temperature: Defines the intersection point of two extrapolated baselines extending from the start point and end point of the absolute running integral of the glass or step transition being analyzed with respect to the current x-axis. This option is primarily used to disregard the effects of enthalpy recovery from the calculation.

Change: Selects how you want the step change (height) to be determined in step transition calculations. The step change is calculated as the change in the Y-axis value between the two designated points. Choose from the following:

  • Start: The step change will be computed using the specified start limit on the Y-axis.
  • Onset: The step change will be computed using the calculated step transition onset point on the Y-axis.
  • Midpoint: The step change will be computed using the onset tangent or end tangent Y-axis value determined at the midpoint X-axis value.
  • End: The step change will be computed using the calculated step transition end point on the Y-axis.
  • Stop: The step change will be computed using the specified stop limit on the Y-axis.

Peak Integration

Use the Peak Integration options to calculate and report the following:

  • Peak onset time or temperature
  • Peak maximum time or temperature
  • Peak area
  • Extrapolated peak

The peak is integrated with respect to time between the designated start and stop baseline limits. You select type of baseline used from the Analysis option (see below). See Basic Steps to Analyzing Data to select associated analysis options before beginning.

  • NOTE: To perform an Peak Integration analysis with divided areas, see TRIOS software Help topic Peak Integration – Divide Peak.

Peak Integration Analysis Options

Baseline Type – Linear

Linear Type

  • Double point: Baseline is determined by two limits that define the transition range on either side
  • Horizontal (from left): Baseline is determine by a linear extrapolation from a point chosen on the left side of the peak
  • Tangent (from left): Baseline is determine by a tangent line drawn from a point chosen on the left side of the peak
  • Manual tangent (from left): Baseline is determine by a tangent line defined by two points chosen on the left side of the peak
  • Horizontal (from right): Baseline is determine by a linear extrapolation from a point chosen on the right side of the peak
  • Tangent (from right): Baseline is determine by a tangent line drawn from a point chosen on the right side of the peak
  • Manual tangent (from right): Baseline is determine by a tangent line defined by two points chosen on the right of the peak

Onset Type

  • Tangent: Uses the selected point to calculate a tangent line
  • Manual tangent: Allows you to define the start and stop limits for the inflection tangent line
  • Inflection: Automatically calculates the onset line as the tangent to the curve at the inflection point (defined as the steepest 1/10 portion of the curve from peak start to peak max).

Baseline Type – Sigmoidal

A sigmoidal baseline is an s-shaped line that changes in level and/or slope before or after a peak. It is used to compensate for the change in baseline that occurs during a transition; the baseline is adjusted for the fraction reacted (alpha) versus time.

A sigmoidal baseline initially is calculated as a straight line between the selected limits. It is then recalculated for each data point between the limits as the weighted average between the projected horizontal or tangent baselines at peak start and end. The weighting factors for a given point are: (1) one minus alpha times the initial baseline and (2) alpha times the final baseline. The area is then recalculated with the new baseline. If the new area differs from the previous area by more than one percent, the area is recalculated and the sigmoidal curve shifted repeatedly until two consecutive calculations of the area differ by no more than one percent. If the baseline fails to converge within ten iterations, the calculation is aborted.

Left/Right Limits

  • Horizontal: To calculate a sigmoidal horizontal baseline, two lines are drawn between the selected limits, then a sigmoidal curve is drawn with its midpoint halfway between the horizontals. This baseline type is often used when a glass transition and cure overlap.
  • Tangent: To calculate a sigmoidal tangent baseline, two tangent lines are fit within the selected limits on each side of the transition. A sigmoidal curve then joins the tangent lines.
  • Manual tangent: The baselines are determined by tangent lines defined by the two points chosen on each side of the peak. A sigmoidal curve then joins the tangent lines.

Onset Type

  • Tangent: Uses the selected point to calculate a tangent line
  • Manual tangent: Allows you to define the start and stop limits for the inflection tangent line
  • Inflection: Automatically calculates the onset line as the line tangent to the curve at the inflection point (defined as the steepest 1/10 portion of the curve from peak start to peak max).

DSC Peak Integration

The program uses the sample size to normalize the area under the peak and obtain the experimental heat in joules per gram. If the sample size is zero, the heat is expressed in joules. If the Y-axis heat units are calories/sec instead of watts, the area is reported in calories per gram (or millicalories if the sample size is zero). If you want the results reported in mJ, uncheck the Normalized Area option on the Analysis Options > Peak Integration page.

Special Area Analysis

Special area calculations can be performed on a Peak Integration (enthalpy) analysis:

  1. Perform the Peak Integration (enthalpy) analysis.
  2. Click Edit Analysis.
  3. Click the Parameters tab.

  1. Select Special area calculation and then choose Percent Cystallinity, Percent Cure, or Heat Per Mole. Make adjustments to the Standard Heat/Heat Per Mole values, if necessary.

Peak Integration for Instruments Other than DSC

The peak area is normalized for other instrument types only when the signal being integrated is normalized.

  • NOTE: For these instruments, the Normalized Area option has no effect.

The peak onset is calculated as the intersection of the onset tangent line with the calculated baseline. The onset tangent line is fit to the current X-axis type, and can be selected in automatic mode (the default condition) or manual mode (selected using the Analysis Options > Peak Integration page).

  • NOTE: The onset line and onset point are not displayed if the axis scaling is log.
  • NOTE: Use caution when integrating peaks that have been rotated. For more information, see Using Analysis Tools.

If the measured data includes superheating or supercooling (temperature hysteresis of more than 0.25°C), the onset point is determined at the superheating or supercooling point. Superheating and supercooling are relevant only if the X-axis is temperature.

Use the Onset point analysis option to determine the temperature at which a change in the slope of the curve occurs (i.e., the onset point of a transition). The onset point is defined as the intersection of an initial tangent line with a final tangent line.

Oxidation Time/Temperature

Use the Oxidation time/temp option to analyze DSC data from oxidation induction time (OIT) and oxidation temperature (OT) experiments.

  • Oxidation induction time (OIT) is a DSC test method that measures the time to oxidation onset of a sample held at an isothermal temperature and in an oxidizing atmosphere.
  • Oxidation temperature (OT) is a DSC test method that measures the oxidation onset temperature of a temperature-programmed sample in an oxidizing atmosphere.

Running Integral

Use the Running Integral option when you want to determine the area under a peak or the ratio of different materials within a sample. For example, in TGA, this function is valuable for determining the relative amounts of evolved gases in a material. In DSC, it can be used to calculate the percent composition of polymer blends.

  • NOTE: The Running Integral option is not available if the Y4 axis is being used to graph a data signal.
  • NOTE: You must use the Analyze/Integrate Peak/Running Integral analysis to obtain the correct peak area when integrating using any integrate peak option other than Linear. The more general Analyze/Running Integral analysis (described above) uses either a linear baseline or zero baseline (absolute integral) for integration, which will give a different running area than that reported by the Peak Integration analysis with a sigmoidal baseline.

DSC Purity Analysis (ASTM E928)

Purity Analysis can be used to calculate the absolute purity of a sample based on data obtained from the DSC (Differential Scanning Calorimeter) according to ASTM procedure E0928.*

  • NOTE: The DSC Purity Analysis is not recommended for analysis of SDT (DSC-TGA) data. The results will not be as expected.

About Purity Analysis

TRIOS uses the van't Hoff equation to calculate your sample's purity, performing a series of operations using the data file. First, there is an initial scan of the peak to find the baseline, peak area, and peak height, which are needed to calculate are segments. A second scan divides the peak into the number of partial area segments specified on the Purity Options - Partial Area page (25 is the default) to find the TS versus 1/F curve. The segments are equally spaced along the 1/F axis. The first segment starts when the peak height is the specified percent of the maximum height or the input value (10 percent is the default). The last segment ends at the specified maximum percent of the total peak area or the input value (50 percent is the default).

  • NOTE: If the curve does not contain enough points to allow the specified number of segments, the allocation continues until the maximum partial area percent of the total peak is reached.

A correction is made at the end of each segment for the temperature difference between the sample and sample thermocouple (using the onset slope value entered in the analysis options).Then a nonlinear least squares technique is used to approximate the area correction for the TS versus 1/F curve. (This is done to linearize the usual curvature of the line, commonly attributed to undetected melting of the sample before peak start.) A least-squares fit to a straight line is used to calculate the slope and intercept of the TS versus the corrected 1/F curve. The corrected peak area is used to calculate the heat of fusion unless you have opted to used the uncorrected area, or have entered a known heat of fusion for the sample. Lastly, the van’t Hoff equation is used to calculate the impurity level and convert it to the purity.

 

If you use this data analysis program to plot TS versus 1/F, it should result in a straight line with a slope of RT02 X/DHf (melting point depression) and an intercept of T0 (melting point of a pure sample). In actual practice, the plot of TS versus 1/F results in a curve. This curve is thought to be the result of "undetected melting." A correction value is added to the value of F so that a straight line results. The impurity level of your sample is calculated from the slope of the TS versus 1/F line. The heat of fusion is calculated using the total peak area.

For additional information on purity analysis, see R.L. Blaine and C.K. Schoff, eds., Determinations by Thermal Methods, ASTM Special Technical Publication 838, Publication code 04-838000-40.

* For more information go to http://www.astm.org or write to ASTM, 100 Barr Harbor Drive, West Conshohocken, PA 19428-2959, (610) 832-9500.

Guidelines for Data Collection

Be sure to calibrate your DSC using indium at the heating rate to be used for subsequent sample analysis. Correct calibration of your DSC for cell constant and onset slope is extremely important. These calibration constants will affect the accuracy of your DSC Purity results.

Before performing Purity analysis, you will need to gather the data. To obtain the best results from the DSC Purity analysis, use the following optimum experimental conditions for purity determinations:

  •  Sample size = 1.7 mg
  •  Heating rate = 0.5 °C/min
  •  Impurity level < 2 mol%
  •  Data sampling interval = 1 sec/point

Encapsulate the sample as tightly as possible to minimize gradients and/or sample movement.

Slight variations from the above values are acceptable; however, for the most accurate results, you need to maintain as many of these parameters, at the most optimum value possible, during data collection.

Basic Steps for Performing Purity Analysis

  1. Collect your data following the Guidelines for Data Collection.
  2. Obtain calibration values for cell constant and slope at 0.5°C/min. Also obtain the molecular weight for the sample to be analyzed–if you don't know this value, use a molecular weight of 100 g/mol.
  3. Locate and open the desired purity file.
  4. Select Purity Analysis from the drop-down menu on the Function toolbar.
  5. Click Analyze. Confirm the molecular weight (update if needed) and click OK.

  1. Click Edit Analysis > Purity tab to set up the desired purity analysis options.

  1. Click Accept to analyze the data.
  • NOTE: When selecting peak integration limits, be sure to select the left (or leading) well below (usually about 5°C) the apparent onset to melting.

Purity Analysis Options

Method A: Partial Area

Select the Method A option to set up the parameters for performing a partial area purity analysis.

  1. Enter the Number of partial areas(the default value is 25). The number of partial areas is computed by the program and used in the purity calculations. The reciprocals of these partial areas are spaced approximately equal.
  2. Enter the Starting partial area fraction (fraction of peak maximum) as a decimal (the default is 0.10 [10 percent]). The first partial area begins at the left edge of the peak range, and extends to the point where the signal is a specified fraction of the peak height below the baseline.
  3. Enter the Ending partial area fraction (fraction of peak area) as a decimal (the default is 0.50 [50 percent of the total area]). The last partial area ends when it exceeds or equals this fraction of the total peak area.
  4. Select Constrain area correction (c). Enter the upper and lower limits of the area correction factor as the Constraints on area correction factor. The "missing area" correction factor must fall within the specified range (0.0 < Factor < 0.2 is the default). You can force the correction to a specific value by setting the lower and upper limits of the range to the same value; however, it is more often used to force the correction to remain below a selected value (the upper limit value).

    If the correction factor selected by the program is equal to the upper limit, the amount may be inadequate to obtain a good fit to the straight line, and DSC Purity obtained may be incorrect. Check the fit (using the RMS deviation value in the Results Report). If necessary, you can reanalyze the data with a larger maximum area correction factor. A small RMS deviation indicates a good fit to the straight line.
  5. Click Accept to apply these parameters.

 

Method B: Peak Height

Select the Method B option to set up parameters for performing a peak height purity analysis.

  1. Enter the Number of partial areas(the default value is 25). The number of partial areas is computed by the program and used in the purity calculations. The reciprocals of these partial areas are spaced approximately equal.
  2. Enter the Starting peak height as a decimal (the default value is 0.10 [10 percent]).
  3. Enter the Ending peak height as a decimal (the default value is 0.90 [90 percent]).
  4. Click Accept to apply these parameters.
