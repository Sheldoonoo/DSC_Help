Overview

If you are running TRIOS on a Windows 7 (or greater) PC, you will be able to view experiment information when you select File > Open within TRIOS to better locate the desired file(s) using windows operating system functions.

 

  • NOTE: TRIOS library functions are only supported by newer operating systems (Windows 7 or greater.)

Customizing the Column Information for File > Open

To display TRIOS experimental information within the Details view:

  1. Within the File > Open window, select the TRIOS library.

  1. Select the Details view (if not already displayed).

  2. Right-click on one of the column headers and select More.


  3. A list of available parameters displays. Select from the available TRIOS experimental information (preceded by “TA” in the description) and then click OK.

Graph Preview

Use the preview pane to see a preview image of the selected file. The last displayed graph view in TRIOS will be stored and displayed.

 

To enable the preview pane, open Windows Explorer or the File > Open window and click the Preview Pane button in the toolbar .

 

  • NOTE: For data files saved before TRIOS V2.6, a preview image will not be available until the file is reopened and then saved again.

To view the files as thumbnail images, select an icon size from the More Options menu.

Filtering Results in TRIOS Library

Use the TA columns in conjunction with the standard Windows columns within the TRIOS library to filter your results files. Hover your mouse over the column name until an arrow displays, and then make your selection from the list. Multiple filters can be used.

A column currently being used as a filter displays a check mark:

To remove a filter, click the check mark in the column name and then deselect the option from the list.

Adding Directories to TRIOS Library

The TRIOS default data file location is added to the TRIOS library as part of the software installation [C:\ProgramData\TA Instruments\TRIOS\Data]. Add additional directories by performing the following steps:

  1. Using Windows Explorer, browse to the desired file path.
  2. Right-click on the path and select Include in Library > TRIOS If you want to add a network location, select the directory option Available offline.

 

  • NOTE: Verify that the index attributes are enabled for each directory added.
  1. Right-click on the directory and select Properties.
  2. On the General tab, click Advanced. The Advanced Attributes window displays.
  3. Select the Allow files in this folder to have contents indexed in addition to file properties option.

The next time the system indexes, the additional directory will be included.

You can also force a build by accessing the Windows Indexing Options:

  1. Access Control Panel > All Control Panel items
  2. Select Indexing Options
  3. Click Advanced and select Rebuild.
