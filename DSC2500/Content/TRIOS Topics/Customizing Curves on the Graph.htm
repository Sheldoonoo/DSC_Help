Curves Format

Use the functions in the Curves Format dialog box (which is available from the Curves button located in the Format tab > Format toolbar) to customize the visual properties associated with the graph. The Curves Format dialog box can also be accessed by right-clicking on a curve in the graph and selecting Curves Format...

Automatic Formatting vs Manual Formatting

Customization of curves on the graph is performed by selecting either the Automatic Formatting or Manual Formatting options. The main difference between the two is that in Automatic Formatting, the increments are linked to the file, step and/or variable, whereas in Manual Formatting, increments are applied per curve. The functionality available for customization depends upon the option selected.

  • The Automatic Formatting option chooses an appropriate selection of increments based on the graph. You can customize which increment is locked to which particular feature, and then edit the properties on individual curves or change schemes.

    Note that the order of the options in the palette are applied to the variables/steps/files in the order in which you add them to the graph. To lock the increment of a specific curve so that it is not modified when you make a change to the graph, use the Manual Formatting option.

  • In Manual Formatting, the curve properties are still incremented based on the selection made, but there is no linking to file, step, and/or variable. Use Manual Formatting to lock a particular curve's color, symbol, line style, or extra symbol.

    Note that by deleting a variable from the graph, the increment associated with that variable will also be deleted instead of being applied to another curve as it would in Automatic Formatting.

Customizing Color, Symbol, Extra Symbol, and Line Style

After selecting your desired increments, customize the selections using the palette (when Automatic Formatting is selected) or setting the differentiators (when Manual Formatting is selected).

Color

If Color is chosen as an increment for the variable, step, and/or file, then the color palette becomes available. Click the color palette to modify the colors to be used for the curves. In the Colors dialog box, selecting the drop-down arrow for each color allows you to change that specific color.

Note that the order of the colors in the palette are applied to the variables/steps/files in the order in which you add them to the graph. To lock the color of a specific curve so that it is not modified when you make a change to the graph, use the Manual Formatting option.

Symbol / Extra Symbol

If Symbol or Extra Symbol is selected as an increment for the variable, step, and/or file, then the symbol palette becomes available. Click the symbol palette to modify the symbols to be used for the curves. In the Symbols (or Extra Symbols) dialog box, selecting the drop-down arrow for each symbol allows you to change that symbol.

Note that the order of the symbols in the palette are applied to the variables/steps/files in the order in which you add them to the graph. To lock the symbol of a specific curve so that it is not modified when you make a change to the graph, use the Manual Formatting option.

For information on setting the symbol size, density, or adding extra symbols, see Additional Formatting Options.

Additional Options:

  • Standard : Allows you to use a generic set of symbols

  • Digits: Allows you to use numbers as your symbols

  • Letters: Allows you to letters as your symbols

  • Custom: Allows you to use a special set of symbols

Line Style

If Line Style is selected as an increment for the variable, step, and/or file, then the line style palette becomes available. Click the line style palette to modify the line style to be used for the curves. In the Line Styles dialog box, selecting the drop-down arrow for each line allows you to change that line style.

Note that the order of the lines in the palette are applied to the variables/steps/files in the order in which you add them to the graph. To lock the line style of a specific curve so that it is not modified when you make a change to the graph, use the Manual Formatting option.

For information on setting the line thickness, see Additional Formatting Options.

Additional Formatting Options

These options are available in both Automatic Formatting and Manual Formatting.

  • Line thickness: Set the thickness of your lines by selecting 1 through 5. The pixels are multiplied by the number chosen and the changes are applied to all lines. Entering 0 removes the lines and shows only points on the curve.
  • Symbol size: Set the size of your symbols by selecting 8 (smallest) through 24 (largest). These numbers can be compared to the standard font size in most word processing programs. Your selection is applied to all symbols on the graph.
  • Extra symbols shown: Type a number (must be greater than 1) to add extra symbols to each curve on your graph.
  • Symbol density: This is the number of symbols displayed on the curve. Select Coarse (fewest), Medium, Fine, or Every point (adds symbols to every point on the curve). Note that Symbol density does not apply to Extra symbols.

Click Preview to view your changes before committing to them. Click OK to set your selections.

Quick Styles

You can set the Style Selection so that you can easily and quickly apply previously defined formatting to your curves.

  • Default: This option allows you to create and apply a default style for the current file type. After making your formatting selections in the Curve Format dialog box, click Set As Default. The next time you open the Curves Format dialog box, selecting Default from the Style Selection drop-down menu will apply your default formatting to the curves for that file type. The Default style will also be applied whenever a new file of that file type is created .
  • Factory Settings: Select Factory Settings from the Style Selection drop-down menu to apply the factory settings to your curves.
  • Add Current Style: Make your style selections in the Curve Format dialog box. When finished, click Add Current Style to save your style. A User Curve Settings Name dialog box prompts you to name your curve style. Click OK to save the style. It will now appear in the Style Selection drop-down menu for future use.

Saving and Resetting Presentation Selections

  • To save your changes, click Save. Your selected presentation settings are saved as default. Selected presentation settings can also be saved as default by clicking Defaults and selecting Set Presentation as User Default.
  • To revert the presentation settings back to saved user defaults or factory defaults, click Default and select the applicable setting.
