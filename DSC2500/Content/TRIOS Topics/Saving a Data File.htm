Saving a Data File

TRIOS produces and saves a data file after running an experiment on your instrument.

In addition to TRIOS files (.tri), you can open and analyze data files from Thermal Advantage/Universal Analysis, Rheology Advantage, and Orchestrator software packages.

To save an open data file, click the TRIOS button in the upper left corner of the main window. A drop-down menu displays.

  • Select Save to overwrite the original data file with the new data file.
  • Select Save As... to name and save the data file. The Save As... function also allows you to save alternative file formats, such as .rad (Orchestrator compatible) or .ua (Universal Analysis compatible).
  • NOTE: Once the experiment is complete or a file is opened, save changes to the file (such as the addition of analysis results) using the Save or Save As... functions.
