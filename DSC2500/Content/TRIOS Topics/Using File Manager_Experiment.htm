Overview

To open the File Manager – Experiment view, use any of the following methods:

  • Click the TRIOS buttonand select New > Experiment file.
  • Select Experiment tab > Setup.
  • Click Experiment on the File Manager panel.

The Experiment view (shown below) displays the current experiment to be run when the Start button is pressed. An experiment consists of sample information, current geometry, and procedure steps. There will always be an active experiment ready to run, and it will be a copy of the last experiment you ran with the sample details incremented. As soon as an experiment is started, it becomes a results file and will be displayed in the File Manager – Results view. While it is running, a green ball beside the file in the File Manager and on the document tab will indicate it as the ‘active document.’

  • NOTE: The Start button is only illuminated when the instrument is in an ‘Idle’ state and the experiment view is displayed.

Performing General Operations

 Use the File Manager – Experiment view to perform the following functions:

  • Open: Opens one experiment file.
  • Open the default experiment file: Opens the default experiment file, which includes the current geometry, sample blank, and default single step procedure.
  • Copy Last: Copies the current geometry information, sample information, and procedure steps from the last experiment run.
  • New Default: Sets the current procedure as the default (for Copy Last).
  • NOTE: There is always an active experiment ready to run; it is a copy of the last experiment run with the sample details incremented.

Editing an Experiment

The following parameters can be edited for a given experiment:

  • To modify sample details such as fluid density, select TRIOS button > Options and click Material properties.
  • Access the geometry information by clicking the Experimenttab and selecting the geometry from the Geometry ribbon. Any changes made to the geometry parameters will also be made to the current geometry.
  • Modify procedure steps by making the desired change and saving the step. See Using Experimental Procedures for more information.

Printing an Experiment Data File

To print the Experiment parameters of a data file, load a data file and then use one of the following methods:

  • In the Experiment view, right-click and select Print.


  • Click the TRIOS button , hover on Print and then select your Print option: Print, Print Preview, Print PDF, Print PDF All Views, or Page Setup.

Using the Tree Control Bars

Expand and condense the tree control bars using the arrows on the left-hand side of the bar:

  • Condensed: Only the bar and is associated text are displayed. Click here for an example.
  • Expanded: The full dialog and all controls are displayed. Click here for an example.
  • NOTE: To customize the colors of the tree control bars, select View tab > Properties while in the Experiment view.

Parts of an experiment file: 

  • Sample: Displays the name of the sample and any notes that were entered at the time the experiment was run.
  • Geometry (if applicable to instrument): Displays the geometry that was chosen for the experiment, along with settings such as gap, stress constant, strain constant, etc. For more information, see Configuring a New Geometry.
  • Procedure: Displays experiment steps.
