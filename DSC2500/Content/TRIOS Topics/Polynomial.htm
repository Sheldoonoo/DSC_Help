Using This Model

This model is a general-purpose equation that fits a wide variety of data. You can use the Polynomial model to fit x and y axis data that have linear, logarithmic, or mixed scales. The model uses the logarithm of the x or y value automatically when these axes are scaled logarithmically.

Comments

An alternative equation is the Straight Line.
