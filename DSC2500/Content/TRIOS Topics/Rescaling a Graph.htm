The Scaling functions allow you to change the limit points, analysis range, and axis scaling when working in the Graph tab. You can zoom in or out on a specific area of the graph and set specific limits and options. For instructions on defining a data range, refer to Defining a Data Range TRIOS Help topic.

To use the TRIOS scaling functions:

  • Open the desired data file.
  • Select the appropriate graph tab at the bottom of the panel.
  • Select the desired rescale function from the topics below.

Using the Zoom Box

To enlarge a particular portion of the curve for analysis, follow these steps:

  • Position the cursor in the general area you wish to enlarge.
  • Hold down the left mouse button and drag to create a box. Draw a box encompassing the area of the curve to be enlarged. Click here to display an example.
  • Release the mouse. If the area selected is acceptable, zoom in using any of the following methods:
  • Move the cursor inside the box and left-click to zoom the x and selected y axis.
  • Right-click to display the pop-up menu. See the Zoom Box Pop-up Menu section below.

You can zoom in repeatedly to expand a smaller portion of the curve. (Alternatively, the Zoom window can be used to perform further rescaling.)

To Use the Zoom Box Pop-up Menu

After drawing the zoom box, right-click within the dotted lines to display the Zoom Box pop-up menu, which contains the following items:

  • Zoom selected axes: Use to zoom on the selected axis within the chosen region.
  • Zoom all axes: Use to zoom on all axes within the chosen region.
  • Select: Use to zoom on the selected axis within the chosen region.
  • Hide data in selection: Hides the data within the chosen region from view.
  • Hide data outside of selection: Hides the data outside of the chosen region from view.
  • To unhide the data, click Unhide All from the Data toolbar in the Edit tab.

After you select an option from the menu, the graph will be rescaled and a Zoom window is displayed in the lower right corner (default position).

To Manipulate the Zoom Window

Once the graph is rescaled, the Zoom window displays in the lower-right corner of the Graph view. Click here to display an example.

The following rescale functions can also be performed in this window.

  • Moving the Focus of the Zoom: To change the focus area of the zoom, click on the zoom indicator (the blue highlighted region) and drag it to a new area of interest.
  • Changing the Area of Zoom: To resize the zoom area so that a larger or smaller area is displayed, place your cursor on the border of the zoom indicator. A double-headed arrow will appear as seen in the figure below. Drag the border until the desired region is displayed.

 

  • Moving the Zoom Window: To move the Zoom window to another corner on the plot, right-click inside the Zoom window and choose your desired location from the pop-up menu displayed.
  • Hiding/Showing the Zoom Window: To hide or show the Zoom window click on the Hide/Show button .
  • Closing the Zoom Window: To close the Zoom window, select Format tab > Scaling toolbar > All Axes Full .

Using the Scaling Toolbar

The Scaling toolbar on the Format tab contains commonly-used plot scaling functions. These functions are also available on the pop-up menu that displays when you right-click within the plot.

See Also
Using the Graph View

 

Restoring a Selected Axis to Full Scale Range

To restore the selected axis to optimum scale settings and view the whole curve in the plot window, right-click the axis and select Full scale axis. The graph is automatically scaled to the minimum and maximum data values for the selected axis only.

 

Restoring All Axes to Full Scale Range

To restore all of the axes on your graph to their optimum scale settings, select Scaling toolbar > Full Scale All . The graph is automatically scaled to the minimum and maximum data values for all of the axes contained in the file.

Manually Setting the Axis Scaling (Scaling)

If you wish to set specific scaling limits for your plot and choose properties such as scaling type (linear, log, or absolute) and other options, select Scaling toolbar > Manual . For detailed information, see Manually Rescaling a Graph.

 

Other Scaling Functions

To access Exact Scaling X and Exact Scaling Y, right-click the plot (not the curve) and select Zooming.

Exact Scaling X

To plot the axis using the exact range of the x-coordinates found in the data file or rescale zoom box, select Exact Scaling X. Leave this menu item checked and the data will be automatically scaled when applicable. When exact scaling is not selected, the plot axis is drawn with evenly-spaced axis labels that fall on the corners of the plot. This is the default setting.

  • NOTE: This option applies only to the current data file.

Exact Scaling Y

To plot the axis using the exact range of the y-coordinates found in the data file or rescale zoom box, select Exact Scaling Y. Leave this menu item checked and the data will be automatically scaled when applicable. When exact scaling is not selected, the plot axis is drawn with evenly spaced axis labels that fall on the corners of the plot. This is the default setting.

  • NOTE: This option applies only to the current data file.

Stack Y Axis  

Select Stack Y Axis from the Scaling toolbar to adjust the y-axis so that the curves on the axes are stacked, one above the other, with no curves from one axis overlapping the curves of another axis.

Using the Axis Scroll Bars

The Graph View has controls that can be used to quickly adjust the plot limits for the plotted axes on the fly. Each axis has three types of controls—scroll buttons, inside scroll bar, and outside scroll bar.

  • Scroll Button: When using these buttons, the curve scaling is maintained, and the axis and its associated curve is shifted up/down or left/right.
  • Inside Scroll Bar: This is used to adjust the left-right positioning of the curve, keeping the scaling magnitude the same.
  • Outside Scroll Bars: These bars are used to increase or decrease the magnitude of the scaling. Only the selected axis scaling is adjusted.

To use the scroll bars:

  • Click on the desired axis.
  • Left-click on the bar and drag it left or right for the x-axis, or up or down for y-axis. Alternatively, you may also left-click on the gray areas in the scroll bar to shift the plot incrementally.

Rescaling Using the Mouse Wheel

The mouse wheel may also be used to zoom in or out on an area (adjusting both the magnitude of the x and y scaling). To use this feature, click on the desired axis then use the mouse wheel to scroll up or down.

  • Each click up corresponds to one click on the Zoom In button.
  • Each click down corresponds to one click on the Zoom Out button.

A zoom box automatically opens when zooming with the mouse wheel (if one is not already displayed). This feature may be used with a zoom box already opened and visible, but is not available when the zoom box is closed.
