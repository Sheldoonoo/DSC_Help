Configuring the Discovery Instruments for Static IP Addressing

There are three options for configuring Discovery instruments: connecting the instrument to a standalone router, connecting the instrument to a switch, or connecting the instrument to a corporate LAN.

Option 1: Connection of Instrument(s) to Standalone Router

In this configuration, instruments are connected to a standalone router. This option requires that a dedicated a 10 Base T /100 Base TX Ethernet card be installed in the computer (controller) to communicate with the instrument. If you want to communicate with your corporate LAN, a second Ethernet card will need to be installed in the computer.

 

Connecting the Instrument to the Router

  1. DO NOT install the CD included with the router, this is not applicable for our use. Plug one end of the power cord into the power connector and the other end into the AC outlet (100-240 Vac 50/60 Hz).
  2. Power off the instrument.
  3. Locate the Ethernet port on the rear of the Common Cabinet.
  4. Connect one end of the Ethernet cable into the Common Cabinetâ€™s Ethernet port.
  5. Connect the other end of the Ethernet cable to one of the LAN ports on the router.
  6. Power on the instrument.
  7. Check the Ethernet port on the rear of the Common Cabinet. If communication between the instrument and the router has been properly established, a solid green light and flashing yellow light will appear at the port.
  8. Follow the directions in the next section to connect the controller computer to the router.

 

Configuring the Ethernet Card for DHCP

To configure the Ethernet card for DHCP:

  1. NOTE: The Discovery Series thermal instruments support both DHCP and Static IP addresses.

Windows 7 (and Up) Users

  1. Click the Windows start button on the taskbar, then select Control Panel > Network and Sharing Center.
  2. Click the Local Area Connection link. A Local Area Connection Status dialog box displays.
  3. Click the Propertiesbutton. A Local Area Connection Properties dialog box displays.
  4. Select the Internet Protocol Version 4 (TCP/IPv4) and click OK.

 

Windows Vista Users

  1. Click the Windows start button on the taskbar, then select Control Panel > Network and Internet > View network status and tasks.
  2. Highlight the Local Area Connection link for the Ethernet card that will communicate with the instrument. Then right-click and select Properties.
  3. From the General tab, highlight the Internet Protocol Version 4 (TCP/IPv4) and then select Properties.
  4. Enable the radio button selection Obtain an IP address automatically and click OK.

 

Connecting the Computer to the Router

  1. Locate the Ethernet port on the back of the computer.
  2. Plug one end of the Ethernet cable into the computer's Ethernet port.
  3. Connect the other end of the cable to one of the network ports on the router.
  4. Check the Ethernet port on the rear of the computer. If communication between the computer and the router has been properly established, a solid green light and flashing yellow light will appear at the port.
  5. Follow the directions in the next section to connect the controller computer to a LAN for networking capabilities.

 

Connecting the Computer to a LAN (optional)

Before you can connect the controller computer to a LAN, you will need to have already installed an additional Ethernet card in the computer.

  1. Locate the second Ethernet port on the back of the computer.
  2. Plug one end of the Ethernet cable into the computer's Ethernet port.
  3. Plug the other end into the corporate LAN.
  4. Check the Ethernet port on the rear of the computer. If communication between the computer and the LAN has been properly established, a solid green light and flashing yellow light will appear at the port.

Option 2: Connection of Instrument(s) to Switch

In order to connect the instrument to a network, you will need to make the necessary cable connections described below. The instrument and computer are connected to an Ethernet switch.

Connecting the Instrument to the Switch

  1. Locate the Ethernet port on the rear of the Common Cabinet.
  2. Connect one end of the Ethernet cable into the Common Cabinet's Ethernet port.
  3. Connect the other end of the Ethernet cable into one of the network ports on the Ethernet switch.
  4. Check the configuration switches located on the back panel. These switches must be set to off (or in the up position) for the controller to communicate with the instrument.
  5. Check the Ethernet port on the rear of the Common Cabinet. If communication between the instrument and the switch is properly established, a solid green light and flashing yellow light will appear at the port.
  6. Follow the directions in the next section to connect the controller to the Ethernet switch.

 

Connecting the Computer to the Switch

  1. Locate the Ethernet port on the back of the computer.
  2. Plug one end of the Ethernet cable into the computer's Ethernet port.
  3. Connect the other end of the cable to one of the network ports on the switch.
  4. Check the Ethernet port on the rear of the computer. If communication between the computer and the switch is properly established, a solid green light and flashing yellow light will appear at the port.
  5. Follow the directions in the next section to connect the computer to a LAN for networking capabilities.

 

Connecting the Computer to a LAN

Before connecting the controller to a LAN, ensure that a network interface card is already installed in the computer.

  1. Locate the second Ethernet port on the back of the computer.
  2. Plug one end of the Ethernet cable into the computer's Ethernet port.
  3. Plug the other end into the LAN.
  4. Check the Ethernet port on the rear of the computer. If communication between the computer and the LAN is properly established, a solid green light and flashing yellow light will appear at the port.

 

Option 3: Connection of Instrument(s) Directly to Corporate LAN

In this configuration, instruments are plugged directly into corporate LAN. This option requires that a single 10 Base T /100 Base TX Ethernet card (configured with corporate LAN settings) be installed in the controller to communicate with the instrument.

Discovery thermal instruments can be configured as DHCP or Static IP. If you are using Static IP addressing, you'll need to ensure the IP, Gateway, and Subnet Mask are compatible with your company network.

  1. Power off the instrument.
  2. Locate the Ethernet port on the rear of the Common Cabinet.
  3. Connect one end of the Ethernet cable into the Common Cabinetâ€™s Ethernet port.
  4. Connect the other end of the Ethernet cable to the corporate LAN.
  5. Power on the instrument.
  6. Locate the Ethernet port on the back of the computer.
  7. Plug one end of the Ethernet cable into the computer's Ethernet port. Plug the other end into the corporate LAN.

Configuring the Discovery Instruments for Static IP Addressing

Currently, the only supported way to change the network settings on the Discovery instruments is to use the built-in Flash tool.

  1. On the instrument User Interface, press Set View.

  1. When the Select a View window displays, press Instrument Properties.

  1. Now press Network.

  1. The instrument Network Properties window displays for the instrument. The default network setting is DHCP. The active network configuration (DHCP or Static) is indicated by dark green checkmark. In the figure shown below, the instrument is configured as DHCP.

  1. To switch the Discovery instrument's configuration to Static, press the Static button. A graphical keyboard appears so that you can input your static IP Address, Subnet Mask, and Gateway.

  1. Enter the IP Address; ensure that the address is unique.
  2. Enter the Subnet Mask. The Subnet Mask assigned must be the exact same range as all the other instruments so that ALL of the instruments will appear within the TRIOS Connection Dialog.
  3. Now enter a valid Gateway. Note the following:
  4. The 1st octet of the Gateway MUST match the 1st octet of the IP address.
  5. The 2nd octet of the Gateway MUST match the 2nd octet of the IP address.
  6. The 3rd octet of the Gateway MUST be between 1 and 255.
  7. The 4th octet of the Gateway MUST be 1.
  1. When you have finished entering your new addresses, press the icon to hide the keyboard.
  2. Now press OK.

  1. A dialog appears asking if you are sure you want to apply the changes; press Yes to accept. If you want to make a change or cancel the operation, press the button No.
  2. After you press Yes, the Common Cabinet reboots.
  3. After the instrument reboots, the new address should appear on the main display on the main display of the User Interface.
  4. To verify the new IP Address, press Set View on the User Interface.
