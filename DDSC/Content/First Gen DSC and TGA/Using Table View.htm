Discovery Instrument: Using the Table View

The Table View provides a quick summary of all the experimental parameters for the sequence. Additional runs can be created in the Table View by simply appending runs down from a selected row/experiment, thereby duplicating all the experimental parameters. Specific experimental conditions can be readily modified, copied and pasted between fields, or filled down or up within a given column. In the Table View, multiple runs can be quickly created and easily verified as appropriate experimental conditions by the operator. Access the Table View by clicking Running Queue or Design View in the File Manager – Experiments view (shown below).

Editing Table Parameters

To edit a value in a Table View cell, click within that cell. Depending on the parameter, enter or select a new value. Clicking away from that field or pressing Enter on the keyboard saves the new information.

Copying Parameters

The Table View allows for easily copying parameters from run to run using fill down/up functions.

To copy parameters:

  1. Select the desired parameter (cell) to copy. A green box appears in the lower right corner of the cell.

  1. Left-click the green box with the mouse and hold down while dragging the box down, selecting the cells in that column as desired.

  1. Release the mouse when satisfied with the cell range. {This same concept can also be applied with more than one column selected.}

Alternatively, you can also select this option using the right-click pop-up menu.

 Customizing the Table View 

Customizing the Table View is important, as there is a limited viewing area for displaying this information. The parameters (columns) should be customized so that the operator has easy access to the run information that is likely to require modification or verification. In addition to column selection, the column order, column width, and color settings can also be customized.

 

Defining Table Parameters (Columns)

  1. Right-click within the column headers and select Customize Columns.

  1. The window below displays.

  1. Use this window to specify the desired parameters to display, the column header text, and column size. Changes are made real-time so you can see the resultant change.
  2. Specifying Column Parameters: Select the desired parameters to display by checking or unchecking the available options. Close the window when finished.
  3. Specifying Column Text: Click on the Display Name text to edit that parameter. An edit field appears over the text. Enter the desired text and click away from the parameter to save.

  1. Specifying Column Size: Click on the Size value to edit the selected parameter. A scroll bar appears above the selected file. Adjust the size to the desired width. Alternatively, adjust the column width by dragging the ends of the column using the mouse.

  1. Reordering Columns: Click on the desired parameter to relocate. Use the Up and Down arrow keys to move the selected parameter to the desired position.

 

Reordering Columns 

There are two methods to reorder the columns within the Table View: using the Column Selector window or dragging/dropping the column as described below.

Click within the column you wish to move, and hold down the mouse while moving the column to the desired column location. Arrow indicators appear, showing where the column will be upon releasing the mouse.

 

Adjusting Columns Widths

There are two methods to adjust the columns widths within the Table View: using the Column Selector window or adjusting the column width using the mouse as described below.

  1. Position the mouse pointer between the columns. A double arrow mouse pointer displays.

  1. Click the mouse and hold down while adjusting the column width to the desired size. Release the mouse when satisfied with the column width.

 

Locking Columns in Place

If the Table View is configured such that a scroll bar is needed to view all the information, you may find it useful to lock some columns in place. These locked columns are always visible while you scroll.

To lock a column, click on the pin symbol on the column header. To unlock, click again. Any locked columns relocate to the top of the column order.

 

Defining Table Properties

Individual components within TRIOS are customized through the Properties panel. To access this panel:

  1. With the desired Table View displayed, click View tab > Properties.
  2. The associated settings for the selected items display.
