Connecting Discovery Instruments to the Dual Common Cabinet

The Common Cabinet in the Discovery thermal series can be used to control up to two instruments when fully configured with an optional power supply/furnace control module to optimize the bench space use. When running in this configuration, consult the topics below for additional information.

  • see "Instrument Configuration and Input Voltage Requirements for Dual Instrument Control"
  • see "Connecting the Instruments to a Common Cabinet "
  • see "User Interface Operation (if applicable)"

Instrument Configuration and Input Voltage Requirements for Dual Instrument Control

When operating two instruments from a single common cabinet (e.g., with two universal power supplies), the minimum input voltage required varies based on instrument configuration. Consult chart below for requirements.

Requirements
Dual Instrument Configurations Minimum Input Voltage Requirements
Discovery DSC / Discovery DSC ≥100 Vac
Discovery DSC / Discovery TGA ≥120 Vac
Discovery TGA / Discovery TGA ≥200 Vac

Connecting the Instruments to a Common Cabinet

During installation of your equipment a TA Service representative will install the system and make the necessary connections for dual instrument control. However, in the event you need to move the equipment and reconnect the hardware, please consult the instructions below.

  • NOTE: Before disconnecting any cables, ensure that the power to the common cabinet is OFF.

 

There are two power supply modules in the common cabinet when operating in dual instrument mode. The top cable connections are for one instrument and the bottom set is for a second instrument.

  • NOTE: The low voltage port and the furnace power cable port are arranged in pairs on the back panel of the Common Cabinet. These two cables must be connected to the same Discovery instrument in order for the system to function properly.

  1. Connect the low voltage cable from the CONTROL OUT 1 port on the back panel of the Common Cabinet to the CONTROL IN port on the back panel of Discovery instrument 1. Repeat step for Discovery instrument 2.

  2. Connect the furnace power cable from the FURNACE POWER OUT 1 port on the back panel of the Common Cabinet to the FURNACE POWER IN port on the back panel of Discovery instrument 1. Repeat step for Discovery instrument 2.
  3. Connect the USB cable from any USB “A” port on the back panel of the Common Cabinet to the USB “B” port on the back panel of Discovery instrument 1. Repeat for Discovery instrument 2.

 

User Interface Operation (if applicable)

Operation of each instrument is separately controlled through the instrument User Interface. The window size can be controlled via the window adjustment icon in the upper right corner of each of the window panels. These windows may be configured to be split (one half for each instrument) or full screen for a single instrument. In any of these cases, the status for both instruments is always shown.

 

 

Select from the following topics for more information:

  • Changing panel size between half and full screen

In the upper right corner of the status line of each instrument is the Panel Adjust Icon. Depending on the current state of the panel the arrow display will change to indicate the action taken upon clicking on the icon. Use these buttons to change from full screen, half screen, and hidden.

  • Swapping instrument positions on screen

Depending on the order the instruments come online, the positions of the instrument panels may not be same each time after the common cabinet is rebooted. The location of the panels can be swapped, if desired, following the instructions below.

  • From the SET View menu on the user interface, select Properties.
  • Within this window click the Switch panel buttons to swap the instrument positions (top to bottom and vice versa).
  • Change instrument name for easy identification of each instrument

The displayed instrument name in the status line may be customized. This may be particularly useful when running in dual instrument configuration.

  • From the SET View menu on the user interface, select Properties.
  • Within this window enter the desired name (e.g., DSC 1234) and click the checkmark to save.
  • Repeat process for the other instrument.
