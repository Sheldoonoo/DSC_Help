Overview

The History View allows you to view information about any experiments that have already been completed and saved. When an experiment completes, the file header containing all of the run information (exclusive of the actual digital data) is saved within a database table, which is stored on the instrument flash disk. This information can be queried to find historical data, used to recreate experiments, or used to open recent data files in the Results Manager for analysis.

The History View database table contains a record of every experiment ever run by the instrument. This table can be queried based on any of the available experimental parameters such as; records within a certain date range, or experiments made with a particular sample name, etc. Once the records have been located, the data can be sent directly to the Results Manager for data analysis or sent to the Design view or Running Queue where they can be modified or rerun.

If you are running TRIOS on a Windows 7 computer, you will be able to view experiment information stored on the computer TRIOS library as well (similar to when you select File > Open) to better locate the desired file(s) using windows operating system functions:

  1. Select the History View.
  2. Click on the Select Instrument icon (shown below) and select Local PC.

 

    • NOTE: TRIOS library functions are only supported by newer operating systems (Windows 7 or greater.) For more information see Using the TRIOS Library.

    • Note that TRIOS software allows for more than one client to be connected to the Discovery Instruments and, as a result, it is important to be aware of the following when opening saved data files:

    • If using the History View to open files, ensure that the data file to be opened was originally saved on the local client opening the file. If the file you are trying to open was not originally saved on the local client, then right-clicking and selecting Open in Results Pane opens a duplicate copy of the original raw data file.
    • Any data files that are analyzed and saved on other clients are also unavailable to open from within the History View.
    • Using the File > Open function from the TRIOS button opens the selected data file normally, regardless of which client is opening the file.

    Filtering Records

    There are several ways of filtering the data within the History View:

    • Using the File Manager calendar, click on the day that you would like to filter. To select more than one date, hold down the Shift key on the keyboard while selecting multiple dates of interest. Note that the maximum selection is two concurrent calendar months.
    • To enable filtering, click the drop-down arrow next to Reset Filter, and then select one of the following filter options: Filter in Row or Filter in Header.

     

    Filter in Row

    The Filter in Row option inserts a row above the first record which can then be used to filter the data. To filter the data, select the filter criteria within the cell and then either click the drop-down arrow and select an item from the available list or manually enter some text. To change the filter, click the clear criteria filter icon or remove the text from the filter field. To completely remove the filter, click the drop-down arrow next to Reset Filter and then deselect the selected filter option.

     

    Filter in Header

    The Filter in Header option inserts filter icons within each of the column headers which can then be used to filter the data. To filter the data, click the filter icon and then select one of the items from the available list. To disable or change the filter, simply click the filter icon and select another filter. To completely remove the filter, simply click the drop-down arrow next to Reset Filter and then deselect the selected filter option.

     

    Filter by Dragging and Dropping Column Header

    A third filtering option is to drag one of the column headers into the field labeled Drag a column header here to group by that column. The records are sorted based on the selected column header; click the + sign to expand the filtered data. To change or remove the filter, drag the header back onto the orange highlight bar and then repeat the above procedure.

    Querying Records

    There are several ways to query data within the History View:

    • Text Search: To query the entire database by text, enter text within the Text Search field. The records are sorted based on the entered search criteria.
    • Refined Search: Another way to query the data within the History View database is to click the drop-down arrow next to Reset Filter, and then select the option Refined Search. This option adds a search bar to the ribbon where you can specify how to refine your search. To define your search, click the drop-down arrow and select one of the following predefined search criteria: Date Time, Mode, Operator, Pan Type, Procedure Name, or Sample Name. Once you select the initial search criteria, you can further refine the search by adding additional search criteria. To add additional criteria, just click the drop-down arrow for each newly added selection, and then select from the available list, repeating the process until you are able to locate the desired records.

    Calibrations

    The History View also allows you to analyze calibration data files directly without having to open the files in the Results Manager. To analyze any of the calibration data files, just highlight and right-click the selected file(s) and then select Calibrations from the right-click pop-up menu. The corresponding analysis option is enabled for the appropriate calibrations, and then an analysis document is generated. At this point, the analysis results can be adjusted or saved and applied. For more information, see the appropriate calibration topic for your instrument.

    Displaying Procedure Information

    The Procedure Information (the method segments that were used to acquire the data) can be viewed for any of the records saved within the database.

    To view the Procedure Information for a specific record:

    1. Locate the record of interest. See Querying Records for more information.
    2. Highlight and right-click the record of interest, and click Show Procedure Info from the pop-up menu.
    3. A control panel for the Procedure Information opens, displaying the method segments that were executed to generate the data for the specified experiment.

    Creating a New Experiment Based on a Previously Run Experiment

    Within the History View, you have the ability to generate a new experiment based on a previously run experiment. This new experiment can be sent to either the Design View or the Running Queue, with the option to send with or without the sample information.

    To generate a new experiment based on a previously run:

    1. Locate the record of interest. See Querying Records for more information.
    2. Highlight and right-click the record of interest, and click the option Send to Running Queue or Send to Design View with or without Sample Information.
    3. When the experiment is sent to either the Design or Running Queue, a conformation message is posted asking if you want to switch to the Experiment view. Click Yes to switch to the Experiment View or click No to remain within the History View.

    Opening Records in the Results Manager

    To open any records from within the History View:

    1. Locate the record of interest. See Querying Records for more information.
    2. Highlight and right-click the record of interest, and click Open in Results Pane. To open multiple records, hold down the Shift key to select all the records or hold down the Ctrl key to selectively choose the files to open.
    3. After selecting Open in Results Pane, the view switches from the History View to the Results Manager, where all of the selected file(s) appear. At this point, you are free to modify, analyze, and save your results. For more information, see Basic Steps to Analyzing Data.

    Copying Data Files

    Data files can be copied from within the History View to any desired location by right-clicking on the selected record and then selecting the option Copy the Selected Data Files.

    To copy any records from within the History View:

    1. Locate the record of interest. See Querying Records for more information.
    2. Highlight and right-click the record of interest, and click Copy Selected Data Files. A Browse for Folder dialog window appears. Navigate to the desired directory and then click OK to complete the operation. To copy multiple records, hold down the Shift key to select all the records or hold down the Ctrl key to selectively choose the files to copy.
    3. When the copy operation has completed successfully, a conformation message appears. Click OK to acknowledge the message.

    Customizing the History View Columns

    The layout of the table within the History View is totally customizable, from the columns that appear in the table to the order in which the columns appear.

    To customize the layout of the table columns within the History View, right-click within the window and select the option Customize Columns. The Column Selector dialog appears. Items in this list will vary based on your instrument.

    Click the checkbox next to each of the items that you would like to appear in the table and uncheck any of the checkboxes of the items that you want to remove from the table view.Change the order of the columns by using the arrows. Once you have made your selections, click the red "x" in the upper-right-hand corner to save and close the dialog.

    Customizing Table View Properties

    The fonts and colors for the background, rows, cells, and headers can also be adjusted within the History View.

    To adjust the fonts and colors, right-click within the History View and select the option Properties. When the Table View control panel opens, you can make changes to the default Table View settings. These changes can be permanently saved to a file and then reloaded at a later date, if necessary. To save any changes to the properties, click the icon. To load any previously saved property file, click the icon and then select the desired file. If you would like to reset all of the settings back to default, click the icon . If you just want to reset the selected object back to default, click the icon.
