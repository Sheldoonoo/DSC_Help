Calibrating the Discovery Common Cabinet User Interface

The instrument user interface may occasionally need calibration to adjust the pressure spots on the screen for proper alignment. Follow these steps to calibrate the user interface:

  1. Select More on the user interface screen, then press Calibrate Screen.

  1. The user interface calibration screen displays. Press each calibration target with the tip of your finger.

  1. Press Accept to accept the calibration or Cancel to quit without saving.
