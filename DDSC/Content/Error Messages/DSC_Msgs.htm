The autosampler failed to find the arm up/down position (502)

This problem could be caused by one of several things:

  • The up/down autosampler motor failed.
  • The lead screw is rotating.
  • There is a problem with the ribbon cable.
  • There is a problem with the up/down sensor.
    The sensor flag is missing or it is not adjusted properly.

All of these things must be diagnosed and corrected by a qualified TA Instruments service representative. Before you call, try the following:

Reset the autosampler by clicking the RESET icon, in the Autosampler section of the Control Panel in TRIOS, or by pressing the RESET button on the instrument keypad.

If the problem persists, contact TA Instruments for service.

DSC Cell Errors

 

Message Text Explanation of Message Solutions to Error
Cell load firmware failed There was a problem while trying to update the cell's firmware. Call TA Instruments for service.
Cell temperature below ambient. Equilibrating to ambient temperature before opening lid This is just a warning indicating that the LID is being opened while the cell temperature is below ambient. This situation can result in frost forming within the cell. To avoid frosting, equilibrate to ambient temperature before opening LID.
Cell temperature failed to equilibrate The run has stopped and data has been saved. The cell could not reach the desired temperature within 90 minutes from the start of an Initial, Equilibrate, or Increment segment. Either the cell temperature is more than 4.0°C from the set point or the cell temperature is changing by more than 0.50°C/min.

Try one or more of the following operations to solve the problem, then restart your experiment.

  • If the cell could not get hot enough, check that the cell lids are in place and that the heater is not burned out. Note that ambient temperature may be affecting the cell.
  • If the cell could not get cold enough, check that the LN2 Pump has an adequate supply of liquid nitrogen and that the LN2 Pump lines are not blocked.
  • If the cell could not get cold enough, make sure that you have not exceeded the limits of the cooling device.
  • If the cell temperature is too low and a cooling device is in use, the cooling device may be overpowering the heater. Try using less coolant or removing or turning off the cooling device.
  • If the cell temperature is too high and a cooling device is not in use, consider installing a cooling accessory or using air cool. A cooling accessory is always required for equilibration below ambient temperature.
  • If there is temperature oscillation around the setpoint, make sure that ambient temperature changes are not affecting the cell (i.e., install all cell lids). Check cooling devices for proper operation.
Cooling ramp specified may exceed cooling capacity. Run continuing. A warning indicating that the running experiment is calling for cooling rates that exceed the capacity of the selected cooler.

Try the following:

  • Select the cooler type in TRIOS Options to match the actual cooler being used.
  • Modify the experiment to use cooling rates more appropriate for the cooler type being used.
  • Allow run to continue although the cooling ramp rates will not meet those expected.

Generally, PCA experiments are performed in the temperature range –50 to 80°C using the standard Extended Range Liquid Light Guide provided with the unit. A quartz light guide is available as an option and must be used for experiments above 80°C. The range with the quartz light guide is –50 to 250°C.

Instrument temperature limits exceeded. Run terminated.(66)

One of the thermocouples in the instrument is indicating a temperature that is outside the instrument's operating limit. The following signals are checked:

  • Tzero temperature °C
  • heater temperature °C
  • flange temperature °C
  • heater temperature mV
  • Tzero mV

Try one or more of the following:

  • Try running the sample in an inert atmosphere and/or to a lower temperature. If the sample starts to burn during an experiment, it can raise the temperature beyond the limit.
  • Turn the RCS or LNCS on at least 20 minutes prior to starting the run.
  • Check the thermocouples to make sure they are clean and that they are showing reasonable values on the RealTime Signal Display screen.

If the problem persists, call TA Instruments for service.

Flange over temperature. Run terminated. (67)

The flange temperature is outside the instrument's operating limit. This can occur under the following circumstances:

  • If a sample starts to burn during the experiment and raises the temperature beyond the limit.
  • If the cooler is not on, or hasn't been on long enough to equilibrate (e.g., flange temperature is too high).
  • If the cooler is not operating correctly (e.g., has run out of liquid nitrogen).

Try the following:

  • Try running the sample in an inert atmosphere and/or to a lower temperature.
  • Turn the cooler on at least 20 minutes prior to starting the run.
  • Check the operation of the cooler.

If the problem persists, call TA Instruments for service.

Bad temperature reading. Hardware error. Run terminated. (81) The difference between the heater temperature thermocouple and the sample temperature thermocouple is too large.

Differences between the two thermocouples typically indicate an internal hardware error.

Send a Standby command to the instrument and try again.

If the problem persists, call TA Instruments for service.

If the problem persists, please contact TA Instruments for service.
