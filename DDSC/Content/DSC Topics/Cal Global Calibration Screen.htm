Overview

The Calibration Data window provides the following functionality:

  • Displays the current calibration set that will be applied to future experiments
  • Allows you to select or unselect the various calibration sets
  • Facilitates creation of new calibration experiments
  • Provides a printable calibration summary

To access the Calibration Data window, select Calibration from the File Manager, then double-click Calibration Data. The following window displays.

Calibration Data Set Summary

A DSC calibration set consists of the following calibrations (shown based on dependency of other calibrations):

  • Tzero Calibration
  • Cell Constant
  • Temperature Table
    • Standard Heat Capacity (Tzero only)
    • MDSC Heat Capacity

    The Calibration Data window provides a summary of the calibration set that is currently applied and the parameters associated with each calibration. To display additional information, click the arrow in the right corner of each summary.

    Changing the Calibration Data Set

    In most cases, you will use the latest calibration. However, if you perform multiple experiments using different heating rates and purge gases, for optimum results you should perform the calibration runs using the same conditions. The Discovery DSC facilitates this functionality by storing all the calibration results in a database internal to the instrument. This allows you to pick and choose your calibration parameters accordingly to match the specific parameters of your experiment.

    To select the calibration set, follow these instructions:

    1. Access the calibration component you wish to change.
    2. Select the desired calibration results using one of the following options:
    • Click on the header to display a list of available Calibration results for selection.

    • Click on the selection bar (blue calibration bullets circled below - not available for Tzero calibration). The selection bar displays the last 10 results. To view later entries, use the page navigation arrows to the right of the selection bar.

    • Select Latest to automatically apply the latest calibration results.
    • Select Not Selected to unselect any calibration set.
    1. As you change selections, the associated information displays.

    1. Click Apply to activate the calibration set, or
    2. Click Cancel to retain the last selection.
    3. NOTE: When you change the Tzero calibration, the other calibrations automatically adjust. If Latest is selected and there is an associated calibration based on the selected Tzero calibration, then it is also selected. If Latest is not selected or there is not an associated calibration, then None is selected for all the other dependent calibrations.

    Creating New Calibration Experiments

    The Calibration Setup screen facilitates creation of calibration experiments. To create a new calibration, follow these instructions:

    1. To access the Calibration Experiment Setup window, select Calibration from the File Manager and then double-click Calibration Setup. The Calibration Experiment Setup window displays. Note that Tzero calibration requires recalibration of all the calibration components.
    2. Select the calibration type(s) you wish to perform by selecting the applicable boxes. Additionally, select Cell Conditioning and/or Baseline Conditioning , if required. A calibration template displays, requesting the necessary experimental parameters associated with the selected calibration.
    • Tzero New Calibration Setup
    • Enthalpy (Cell Constant) New Calibration Setup
    • Temperature Table New Calibration Setup
    • Standard Heat Capacity New Calibration Setup
    • MDSC Heat Capacity New Calibration Setup
    • Once the requested information is entered, select from one of the following options. For additional information on auto calibrations and scheduling run sequences, refer to DSC Calibration Overview.
    • Run All: Sends the calibration experiment to the Running Queue and executes the run sequence.
    • Queue All: Sends the calibration experiment to the Running Queue for later execution.
    • Schedule All: Allows you to specify how often the calibration should be run, the time, and the begin and end date.
    1. Once the experiments are complete, you can analyze, save and apply the new calibration.
    • Tzero Calibration Analysis
    • Enthalpy (Cell Constant) and Temperature Calibration Analysis
    • Standard Heat Capacity Calibration Analysis
    • MDSC Heat Capacity Calibration Analysis

    Printing the Calibration Summary

    To print out a summary of your global calibration window, select Print from the right-click popup menu.
