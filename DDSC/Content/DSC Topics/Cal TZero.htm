Overview

The DSC Tzero* calibration requires two experiments. The first experiment is done without samples or pans (baseline); the second is performed with large (approximately 95 mg) sapphire disks (without pans) on both the sample and reference positions. Both experiments use the same method beginning with an equilibration at an initial temperature, followed by holding isothermal for 10 minutes, heating at constant rate to a final temperature and holding isothermal for 10 minutes. The range of temperatures should be at least as broad as the expected range of subsequent experiments. Tzero calibration should be done at relatively high heating rates such as 20°C/min in order to obtain the most accurate calibration of the sensor thermal capacitance and resistance values. Rates of less than 10°C/min are not recommended for Tzero calibration.

* Tzero™ describes proprietary technology invented by TA Instruments-Waters LLC and patented by TA Instruments-Waters LLC (U.S. Patent No. 6,431,747).

 

Recalibration is recommended if the type of purge gas or the cooling accessory has been changed. Recalibrating periodically is also recommended, especially if quantitative performance has degraded over time.

Follow these basic instructions for Tzero calibration:

  1. Collect Tzero calibration data, which requires two experiments—one run without samples or pans and one run with sapphire disks on the reference and sample positions (without pans).
  2. Analyze the data.
  3. Apply calibration results.

Collecting Tzero Calibration Data

The Tzero calibration experimental setup can be performed by setting up the experiments manually using the Experiment view or automatically using the New Calibration Setup from the Global Calibration Set page. The New Calibration Setup method is the recommended procedure since it facilitates multiple run setup and simplifies input parameters.

You can collect data in one of two ways:

  1. Using the Calibration Setup
  2. Using experimental procedure templates

 

Tzero Calibration Using New Calibration Setup

  1. Access the File Manager and select the Calibration panel. Then double-click Calibration Setup.

  2. When you create a run from the Calibration Experiment Setup page, the settings are automatically saved. The next time TRIOS is started, you will be asked if you would like to use your saved settings. Click Yes to populate the fields with your previously entered data, or No to receive a blank experiment setup page.

  3. The Calibration Experiment Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.
  4. Select the Tzero checkbox to open the calibration form.

  1. Enter the following information.
  2. Enter the Name of the new calibration.
  3. Enter the Operator name.
  4. Enter the Project name, if desired.
  5. Enter Sample pan position for sapphire disk (applicable to second run) along with the sapphire mass.
  6. Click the Reference field to display the drop-down table. Select the pan position for sapphire disk (applicable to second run) along with the sapphire mass. If the pan position and sapphire mass are not in the table, click Edit at the bottom of the table to enter applicable values.
  7. Enter any comments or Notes, if desired.
  8. Modify method parameters, if required.
  • Enter the Ramp rate, Lower temperature and Upper temperature limits, and Isothermallimit. The range of temperatures should be at least as broad as the desired experimental range of subsequent DSC experiments.
  • Tzero calibration should be done at relatively high heating rates such as 20°C/min in order to obtain the most accurate calibration of the sensor thermal capacitance and resistance values. Rates of less than 10°C/min are not recommended for Tzero calibration.
  • Select Verification to perform a baseline verification experiment after the Tzero calibration is applied. Note that there is no verification criteria associated with this experiment and that it is for manual inspection only.
  1. When complete, select from one of the following options:
  • Run All: Sends the two calibration experiments to the Running Queue and execute run sequence.
  • Queue All: Sends the two calibration experiments to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Tzero Calibration Setup Using the Template

This calibration requires two experiments—one experiment without samples or pans and one experiment with sapphire disks on the reference and sample positions (without pans). The basic procedure for Tzero calibration is displayed below. The experimental setup follows the same general instructions as a standard experiment; (see Defining DSC Experimental Parameters).

To set up a Tzero calibration:

  1. Access the Design View and add a run.
  2. Click on the associated run form.
  3. Click on the Sample bar. Enter the required test parameters for the first Tzero experiment.
  4. Enter zero for both the sample and reference pans as well as for the associated sample sizes.
  5. Note that the Name field varies based on the procedure mode/test type selected below. For Tzero calibration, this field is not programmed.
  6. Click on the Procedure bar. Enter the test parameters.
  • Select Calibration as the Mode.
  • Select the Test type Tzero. The associated procedure template displays. Enter the requested information. Select the Segments tab to view the associated method in segment form.
  • Lower Temperature & Upper Temperature: Calibration should be performed using the same atmosphere and cooling accessory that will be used in subsequent experiments. The selected lower and upper temperatures for calibration must be within the operating range of the selected cooling accessory and the temperature range for calibration should be as broad as the range that will be used on your samples in subsequent experiments. Typical temperature ranges are: –90 to 550°C for the RCS90, –40 to 400°C for the RCS40, and 50 to 400°C for the FACS. Note that the Quench Cooler cannot be used for Tzero calibration and experimentation.
  • Ramp: A ramp rate of 20°C/min is recommended.
  • Right-click within the Design View and select Append Run. The appended run highlights.
  • Click on the Sample bar. Enter the requested test parameters for the second Tzero experiment (the run appended in the previous step).
  • Enter the Sample position # and sample sapphire weight.
  • Enter the Reference position # and reference sapphire weight.
  • Click on the Procedure bar. Verify that the procedure information matches the first Tzero run.
  • Once the two experiments are setup, copy the experiments in the Design View to the Running Queue and select Start to begin the calibration runs.
  • Once both runs are complete, analyze the data. See Analyzing DSC Tzero Calibration Data.

Analyzing Tzero Calibration Data

Once the calibration experiments complete, analyze the data using the instructions below.

  1. Open the two calibration files in TRIOS.
  2. Select the two files in the File Manager Results pane.
  3. Select Calibrate > Tzero from the Instrument menu options. Alternatively, you can select this option from the File Manager right-click pop-up menu.
  4. When you select Calibrate > Tzero, a new document is created that contains the associated calibration results.

Several views are available within this document.

  • Spreadsheet: A table view of the associated calibration results (Capacitance and Resistance values)
  • Graph: A graphical view of the associated calibration results.
  • Results: A summary of the calibration parameters.
  1. Before saving the results, view the calibration graph. The profile is unique to each cell, but the graph should have the following general characteristics. If the profile doesn’t meet the criteria below, repeat the experiments. If problem persists, contact TA Instruments Service.
  • X-shaped profile with capacitance values decreasing and resistance values increasing
  • Smooth curve shape with no spikes

  1. On the Results view, select from one of the following options:
  • Save: Sends the results to the calibration database, but does not enable or apply the results.
  • Save and Apply: Sends the results to the calibration database and enables the calibration and then applies the results.
  • NOTE: This calibration document will automatically be saved to the following folder: C:\ProgramData\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows 7 or 8, or C:\Documents and Setings\All Users\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows XP.

Selecting Tzero Calibration Results

The Discovery DSC stores all the calibration results in a database internal to the instrument. This allows you to access and set calibrations accordingly to correspond to the specific parameters of any experiment. Selection of the calibration set is made on the Calibration Data page. After analyzing and saving the results, you can apply the results so they are used in future experiments. Access the Calibration > Calibration Datawindow from the File Manager to access the Calibration Data page. Verify the desired calibration set is selected after applying the new calibration result.
