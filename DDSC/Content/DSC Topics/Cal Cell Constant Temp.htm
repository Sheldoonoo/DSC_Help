Overview

Calibrating Enthalpy (Cell Constant) and Temperature

This calibration is based on an experiment in which a standard metal (for example, indium) is heated through its melting transition. The calculated heat of fusion is compared to the theoretical value. The cell constant is the ratio of these two values. The onset slope, or thermal resistance, is a measure of the suppression of temperature rise that occurs in a melting sample in relation to the thermocouple. Theoretically, a standard sample should melt at a constant heating rate. As it melts and draws more heat, a temperature difference develops between the sample and the sample thermocouple. The thermal resistance between these two points is calculated as the onset slope of the heat flow versus temperature curve on the front of the melting peak. The onset value is used for kinetic and purity calculations to correct for this thermal resistance.

The temperature calibration data from this experiment is automatically calculated when the cell constant calibration results are saved.

Calibration data is determined for all the available heat flow types, so this calibration does not need to be redone if you change heat flow selection within the TRIOS > Options > Discovery DSC options.

Calibrating Temperature

Temperature calibration is based on an experiment in which a temperature standard (for example, indium) is heated through its melting transition using the same conditions to be used in subsequent measurements ( heating rate and purge gas). The measured melting temperature of this standard is compared to the known melting temperature and the difference is calculated as the temperature calibration. Up to five unique standards may be used for temperature calibration. A single-point calibration shifts the sample temperature by a constant amount. A two- or more-point calibration shifts the temperature by a constant amount below the first point, uses an interpolated shift function through the calibration points, and shifts by a constant amount after the last point.

During Cell Constant calibration, the temperature calibration data from the cell constant calibration experiment is automatically stored when the cell constant calibration results are saved.

Follow these basic instructions for Enthalpy/Temperature calibration.

  1. Collect Enthalpy/Temperature Calibration data, which requires heating a standard (for example, indium) through its melting transition using the same conditions to be used in subsequent measurements (heating rate and purge gas). The process described here uses only indium for temperature calibration. However, a total of five standards may be used. See Using Multiple Standards to Calibrate Temperature for more information.
  2. Analyze the data.
  3. Apply calibration results.

Collecting Enthalpy (Cell Constant) and Temperature Calibration Data

The Enthalpy/Temperature calibration experimental setup can be performed by setting up the experiments manually using the Experimental View or automatically using the Calibration Setup page. The Calibration Setup method is the recommended procedure since it facilitates multiple run setup and simplifies input parameters.

You can collect data one of two ways:

  1. Using Calibration Setup
  2. Using standard experimental creation using procedure templates

Sample Preparation

  1. Choose an appropriate calibration material with a known heat of fusion (for example, indium).
  2. Weigh an empty sample and reference pan and record their weights.
  3. Prepare a 1 to 5 mg sample of calibration material in a sample pan and record the weight. Crimp the pan, and then place it on the sample platform (or within an available Autosampler tray position).
  4. Prepare a reference pan using the same pan type used for the sample. Crimp the pan, then position it on the reference platform (or within an available Autosampler tray position).
  5. Verify the calibration standard properties (theoretical enthalpy and melting point) within Discovery DSC > Standards page of TRIOS Options.

 

Enthalpy Calibration Using Calibration Setup

Before you begin, verify that a Tzero calibration had been run previously on the instrument, as a Tzero calibration must have been performed, analyzed, and applied before a Cell Constant calibration procedure can be run. If a temperature calibration is run without a Tzero calibration saved on the instrument, you will receive the error below. A Tzero calibration does not need to be performed with the new temperature calibration that is going to be run.

  1. To create a new calibration, access the File Manager Calibration panel. Then double-click Calibration Setup.
  2. When you create a run from the Calibration Experiment Setup page, the settings are automatically saved. The next time TRIOS is started, you will be asked if you would like to use your saved settings. Click Yes to populate the fields with your previously entered data, or No to receive a blank experiment setup page.
  3. The Calibration Experiment Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.
  4. Select the Cell Constant/Temperature checkbox to open the calibration form.

  1. Enter the following information.
  1. Enter Operator name.
  2. Enter Project name, if desired.
  1. Click the Reference field to display the drop-down table. Select the pan position along with the sample mass. If the pan position and sample mass are not in the table, click Edit at the bottom of the table to enter applicable values. If loading pans manually, set this field equal to 0 (None).
  2. Enter any comments (Notes), if desired.
  3. Program the method parameters.
  • If you are using this standard for the first time, check the Premelt option to melt the sample prior to collecting data.
  • Click Add Experiment to add as many experiments as needed. Use the drop-down menu on the Reference to change the reference type. Once an experiment is added, the Remove Experiment button becomes available.
  • Select Calibration to perform a verification run after the calibration completes.
  • Select Verification to perform another calibration in standard mode if the initial calibration run fails. Enter the Verification Criteria (temperature and enthalpy). Note that if the calibration fails, a warning message displays and the experiment will continue with the next item in the Running Queue.
  1. When complete, select from one of the following options:
  • Run All: Sends the calibration experiments to the Running Queue and execute run sequence.
  • Queue All: Sends the calibration experiments to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Enthalpy/Temperature Calibration Setup Using the Template

This calibration requires one experiment using indium. A general procedure for calibration is given below. Experiment setup follows the same general instructions as a standard experiment. See Defining DSC Experimental Parameters. Specific instructions applicable to enthalpy/temperature calibration are outlined here.

  1. Choose an appropriate calibration material with a known heat of fusion (indium).
  2. Weigh an empty sample and reference pan and record their weights.
  3. Prepare a 1 to 5 mg sample of a known calibration material in a sample pan and record the weight. Crimp the pan, and then position it on the sample platform (or within an available Autosampler tray position).
  4. Prepare a reference pan using the same pan type used for the sample. Crimp the pan, then position it on the reference platform (or within an available Autosampler tray position).
  5. Access the Design View and add a run.
  6. Click on the associated Design Run Form (Single Run Form).
  7. Click on the Sample bar. Enter the required test parameters for the experiment.
  8. Click on the Procedure bar. Enter the requested test parameters.
  9. Select the Mode as Calibration.
  10. Select the Test type as Cell Constant. The associated procedure template displays. Enter the requested information. Select the Segments tab to view the associated method in segment form.

  • Premelt: If you are using this standard for the first time, check the Premelt option to melt the sample before starting the experiment.
  • Equilibrate to a temperature below the onset of the known melting temperature of the material. Use the following equation to determine that temperature: 5 min. X Heating Rate (°C/min) = Temperature (°C)
    Typical example: 5 min X 20°C/min = 100°C (below the onset of the literature melting temperature)
  • Ramp: Heat the material at the same heating rate that you will use in your subsequent experiments, to 50°C above the literature melting temperature
  • Once the experiment is set up, copy the experiment to the Running Queue and click Start to begin the calibration run.
  • Once the experiment is complete, analyze the data. See Analyzing DSC Enthalpy/Temperature Calibration Data.

Analyzing Cell Constant/Temperature Calibration Data

There are several ways to analyze the cell constant and temperature calibration data files within TRIOS. If you have multiple calibrations of the same calibration standard, TRIOS automatically calculates the results of each of the individual calibrations, then takes the average of those calibrations and applies that resultant to the instrument. All calibrations, including Tzero, Cell Constant, Temperature and Heat Capacity, both Standard and Modulated, can be analyzed from either the Results Manager or from within the History View.

  1. Access the File Manager and click the Results tab or the History tab.
  2. Right-click and select Open from the pop-up menu to open the calibration file(s).
  3. Highlight all of the file(s) (indium plus all other temperature calibrants, if applicable), then right-click and select Calibrations… > Temperature / Cell Constant, as shown below (left image is the Results tab; right image is the History tab).

  • NOTE: If you are analyzing multiple temperature calibrations, you MUST analyze all of the calibration data files (including indium) simultaneously so that they appear within the same calibration set
  1. A summary of the calibration results is generated in the form of a new Calibration Results document, as shown below. This Calibration Results document offers several different views.

  • Spreadsheet: A table view of the associated calibration results (Measured Temperatures, Offsets, and Cell Constant)
  • Graph: A graphical view of the associated calibration results.
  • Results Summary: This selection displays a summary of the calibration results. If desired, add a description or make changes to the calibration name.
  • Standard: The selected standard
  • Heat Flow Type: Calibration data is determined for all the available heat flow types, so if you change the heat flow selection within the TRIOS Options > Discovery DSC options, you do not need to recalibrate.
  • TMelt (Ref): Theoretical melting point obtained from the Discovery DSC > Standards page of TRIOS Options.
  • TMelt (Meas): Experimental melting temperature per Heat Flow type. Review the associated graph to verify the analysis limits are as desired.
  1. To view or edit any of the individual calibrations, click Edit/Show details.

  1. Click GoTo to view the graph of the calibration analysis, as well as verify the analysis limits.
  2. A new File Overlay document is created showing the analysis.
  3. To adjust the analysis limits for the selected plot, click from the Analysis ribbon.
  4. Using the mouse, drag the cursors to the desired locations and then click Accept in the Edit Analysis dialog box. to save the new limits. If you prefer to enter the limits manually, click Settings, then select the tab Limits and enter your left and right x-axis coordinates.
  5. After making any changes to the limits, the offset, enthalpy, and cell constant are automatically recalculated based on the new limits.
  6. Click the tab Calibration Results to view your results.
  7. If you would like to adjust the limits for any of the other calibration standards or heat flow types, repeat step 7 to step 11.
  8. To save the calibration results, select one of the following options:
  9. Save Results: This option saves the results to the calibration database in the instrument, but it does not enable the calibration.
  10. Save and Apply: This option saves the results to the database and enables the calibration.
  11. When clicking either of the save options, an entry is automatically created on the Global Calibration page in the form of a "calibration bullet". The active calibration is signified by a blue “bullet”.
  12. To view all of the calibrations, access the File Manager and select the Calibration tab. The active calibrations are signified by the blue “bullets”.
  13. To change a calibration, simply select a different calibration bullet, and click Apply to enable that calibration.

  1. After saving the results, a copy of the Results Calibration document is automatically saved to the following folder:
  • Windows 7 and 8, and Vista: C:\ProgramData\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs
  • Windows XP: C:\Document and Settings\All Users\Application Data\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs

Selecting Enthalpy & Temperature Calibration Results

The Discovery DSC stores all the calibration results in a database internal to the instrument. This allows you to access and set calibrations accordingly to meet the specific needs of your experiment. Selection of the calibration set is made on the Calibration Data.

After analyzing and saving the results, you can apply the results so they are used in future experiments.

Access the Calibration > Calibration Data window from the File Manager to access the Calibration Data screen.

Verify the desired calibration set is selected after applying the new calibration result.

Using Multiple Standards to Calibrate Temperature

A total of 5 calibrants can be used for Temperature calibration.

  1. Follow the instructions in Enthalpy Calibration Using Calibration Setup above to create a new Cell Constant calibration using an indium standard. Add the experiment to the Running Queue.
  2. If you wish to add additional Temperature calibration standards, return to the Calibration Data page and again create a new Enthalpy Calibration and send it to the Running Queue.
  3. Repeat step 2 until all the standards needed are added to the Running Queue.
  4. Run the sequence of experiments.
  5. Once the experiments are complete, analyze the data.
