Discovery DSC: Guidelines to Using the LN2P

Once the LN2P has been properly installed (as described in the Liquid Nitrogen Pump (LN2P) Getting Started Guide), follow the steps below to set up the instrument parameters and condition the LN2P system for optimum performance.

  • A dry, moisture-free gas source is required for the cooling gas (LN2P) purge and the base purge when using the LN2P, in addition to the standard cell purge gas. Dry nitrogen is recommended for this purge. These gases must remain on continuously. If they do not, the atmospheric moisture will enter and contaminate the system.
  • By default, the LN2P purge (cooling gas) is off whenever the cell is opened by the AutoLid. It is strongly recommended that the cell lid be closed anytime that a sample is not being actively loaded or unloaded. To activate the LN2P purge whenever the cell is open, access the Discovery DSC > Cooler setting within the TRIOS Software Options window and check Activate secondary purge when lid is opened (RCS and LN2P).
  • Important: If you are planning to run subambient experiments, use helium as the purge gas; in this case, it is recommended to connect helium to GAS 2 and leave nitrogen connected to GAS 1 for base purge. If you are using the LN2P for rapid cooling above ambient (i.e., isothermal crystallization), then nitrogen may be used as a purge gas (GAS 1 can be selected since it uses the same port as the base purge).
  • NOTE: Ensure that you run your experiments with the same gas that you used to calibrate the system. For example, if you calibrate using nitrogen, make your runs with nitrogen.
  • Access the Options > Discovery DSC > Cooler setting window. Verify that the correct cooler type (LN2P) is selected and verify the desired Between Run settings. Enable Leave Cooler On. Check the LN2P Autofill if below option, then enter a percentage to automatically fill the LN2P, if desired. This setting indicates that you want the LN2P to automatically fill when the level of liquid nitrogen falls below the specified percent. The fill process, when activated, will fill to completion. If left unchecked, you will need to manually fill the LN2P when needed. (The default setting is “checked, 40%”.) See Setting Up DSC Cooler Options for more information.
  • When setting up experiments, be sure to verify the post-test conditions. The temperature window should be enabled and a temperature range above ambient should be used to prevent the cell from cooling down between experiments (typical values are 35°C to 50°C). Once these conditions are verified, set the Standby Temp from the Control Panel to invoke the standby temperature specified.
  • When setting up an Autosampler sequence, access the Options > Discovery DSC > Autosampler page and select the desired sequence-end option for the LN2P. Select or deselect your sequence-end Autosampler behavior: Unload Pans and Leave Cooler On. See Setting Up DSC Autosampler Options for more information.
  • To use the local filling capability (autofill), connect the cooling accessory to a low pressure bulk source of liquid nitrogen, then fill the Dewar. See Connecting and Autofilling the LN2P for more information.
  • Important: DO NOT open the DSC cell at below ambient temperatures to prevent frost and moisture buildup in the cell. If this occurs, the conditioning and calibration steps may have to be repeated.
  • Important: Once the DSC-LN2P system has been conditioned, it is recommended that you do NOT turn off the LN2P between runs, if the best possible baseline performance is desired.
  • Important: Operating without an effective base purge, allowing the cell to remain at the lower temperature limit without heater power (e.g., without post-test conditions) for extended periods of time, and/or removing the LN2P from the cell when the flange temperature is below ambient can result in excessive moisture in the cell and requires extended time for drying. Refer to Drying the DSC-LN2P System Help topic for more information.
  • CAUTION: Do not exceed 100°C with the LN2P cooling head installed and the LN2P not operating. Serious damage to the cooling head could occur.
  • CAUTION: TA Instruments recommends that you do not use the LN2P when running isothermal experiments above 400°C. The life of the DSC cell heating element can be shortened if the LN2P is used at high temperatures for extended periods.
  • NOTE: Once the cooling flange reaches operating temperature, it condenses any moisture present. If the initial moisture level is too high, or if the atmosphere moisture subsequently entering the heat exchanger enclosure is not minimized, then artifacts can be observed in the heat flow signals. Typically, but not exclusively, the artifacts are observed between 0 and 100°C, which increase in intensity over time.
