Discovery DSC Autosampler: Full Auto Calibration

This procedure automatically performs the steps needed for calibration of the Autosampler tray and/or cell positions. This is accomplished using a beam of light generated by a fiber-optic amplifier located in the sample arm. The light beam projects downward from the lens in the center of the gripper and the reflected signal transmits back through the same lens and to the fiber-optic amplifier. The entire procedure takes approximately three minutes.

  1. NOTE: It is rarely necessary to perform the calibrations described below. For best calibration results, perform the Verification and Adjustment Procedure.

  1. From the Discovery DSC user interface, access the Autosampler calibration functions by selecting Set View > Autosampler > Calibrate.
  2. Select Options > and press Full Auto Calibration. The auto calibration screen appears.
  3. Select between Tray Calibration and/or Cell Calibration.
  • Tray Calibration: This procedure automatically calibrates the sample tray positions. Before you begin the calibration procedure, make sure the tray calibration start position is correct.
  • Cell Calibration: This procedure automatically calibrates the cell positions using a special cell calibration fixture (shown below). Before you begin the calibration procedure, make sure the cell calibration start position is correct.

  1. Prepare the Autosampler for calibration:
  • If Tray Calibration is selected, grasp the center of the sample tray and pull it straight up. Remove the tray exposing the plate underneath.
  • If Cell Calibration is selected, remove any pans left in the cell. Place the calibration fixture in position over the sample and reference positions.
  1. Select the Run Calibration button. The calibration proceeds automatically. Do not interrupt the procedure. Once this calibration is complete, the Autosampler arm returns to the park position and a calibration complete message appears.
  2. Replace the sample tray and/or remove the cell calibration fixture when process complete.
  3. If more refined calibration is needed after performing this calibration, use the Autosampler Verification and Adjustment procedure.
  4. NOTE: You cannot start experiments when any Autosampler calibration user interface screens are displayed. Select < Back to return to the main Autosampler window.
