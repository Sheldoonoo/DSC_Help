Discovery DSC Autosampler: Cycle Test

After calibrating the Autosampler, you can perform a Cycle Test to test these calibration positions. During the cycle test, the Autosampler loads a sample pan into the cell (beginning with pan number 1), closes the lid, then opens the cell and unloads the pan. Then it proceeds to the next pan number and repeats the operation until the user stops the cycling.

 

The Autosampler Cycle Test functions are used to test the Autosampler pan loading, unloading, tray rotation, and gripper finger functionality. To perform an Autosampler Cycle test, follow these steps:

  1. From the Discovery DSC user interface, access the Autosampler calibration functions by selecting Set View > Autosampler > Calibrate.
  2. Select Options > and press Cycle Test.
  3. Select the desired Cycle Test from the list. A brief description of the function of each item is provided in the table below. To stop the cycle test, press Stop. The Autosampler automatically unloads any pan left in the cell, then returns to the park position. The Cycle Test also automatically stops when it cannot pick up a pan or when there is no pan to pick up.
  4. When finished, select < Back to return to the main Autosampler calibration window.
Cycle Test Functions
Item Description

Cycle Test

 

Choose the desired cycle test from the available list:

  • Cycle all motors: Performs all of the tests listed below for the set number of cycles
  • Cycle arm up/down: Moves the Autosampler arm up, then down for the set number of cycles.
  • Cycle arm left/right: Moves the Autosampler arm left and right for the set number of cycles.
  • Cycle finger: Opens and closes the Autosampler gripper fingers for the set number of cycles.
  • Cycle tray rotation: Rotates the tray for the set number of cycles.

Set

 

 Specify the number of cycles to perform for the specified test. Press the keyboard icon to the right of the Set field to enter a value.
Pan #

 

 Displays the current pan # for the active test (if applicable).
Count

 

 Displays the number of cycles that have been performed for the active test.
Start

 

 Begins the cycle test.
Stop

 

 Stops the cycle test.
