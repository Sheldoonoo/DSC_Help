Overview

Heat capacity can be determined in a single experiment with the Discovery DSC. Heat capacity calibration is for accurate measurement of the sample’s heat capacity. The calibration uses a standard material with a known heat capacity over a specific temperature range of interest (typically sapphire or polystyrene is used). Calibration is typically done at a heating or cooling rate of 10 to 20°C/min. The range of temperatures should be at least as broad as the desired experimental range.

The Standard Heat Capacity calibration curve is calculated by dividing the theoretical value of heat capacity by the measured value over a specific temperature range. The resultant calibration data, as a function of temperature, is sent to the instrument.

Follow these basic instructions for Standard Heat Capacity calibration.

  1. Collect Standard Heat Capacity calibration data, which requires heating a standard (in pans).The experimental range may be limited based on the standard selected. Perform the procedure: The experimental method consists of an equilibration at an initial temperature, holding isothermal for 10 minutes, heating at constant rate to a final temperature and holding isothermal for 10 minutes. The range of temperatures should be at least as broad as the desired experimental range using the same conditions to be used in subsequent measurements (heating rate and purge gas).
  2. Analyze the data
  3. Apply calibration results

Collecting Standard Heat Capacity Calibration Data

The Standard Heat Capacity calibration experimental setup can be performed by setting up the experiments manually using the Experimental View or automatically using the New Calibration Setup from the Global Calibration page. The New Calibration Setup method is the recommended procedure since it simplifies run setup and input parameters.

You can collect data one of two ways:

  1. Using New Calibration Setup
  2. Through standard experimental creation using procedure templates

 

Sample Preparation

  1. Obtain the sapphire disk (P/N 915079.901E), suitable polystyrene sample, or material for which you have theoretical heat capacity data, and record the sample weight.
  2. Weight an empty sample and reference pans and record their weights.
  3. Crimp the standard material into the sample pan, and then position it on the sample platform (or within an available Autosampler tray position).
  4. Prepare a reference pan using the same pan type used for the sample (empty pan). Crimp the pan, then position it on the reference platform (or within an available Autosampler tray position).

 

Standard Heat Capacity Calibration Using New Calibration Setup

Before you begin, verify that the appropriate Tzero and Cell Constant/Temperature calibrations had been run previously on the instrument, they must have been performed, analyzed, and applied before beginning the Standard Heat Capacity calibration procedure. If a standard heat capacity calibration is run without Tzero and Cell Constant/Temperature calibrations saved on the instrument, you will receive the error below. These calibrations do not need to be performed with the new heat capacity calibration that is going to be run.

  1. Access the File Manager and select the Calibration panel. Then double-click Calibration Setup.
  2. When you create a run from the Calibration Experiment Setup page, the settings are automatically saved. The next time TRIOS is started, you will be asked if you would like to use your saved settings. Click Yes to populate the fields with your previously entered data, or No to receive a blank experiment setup page.
  3. The Calibration Experiment Setup screen displays, allowing you to generate new calibrations. To view current calibration sets, open Calibration Data.

  4. Select the Heat Capacity checkbox to open the calibration form.

  1. Enter the following information.
  2. Select the standard from the Name drop-down menu.
  3. Enter the Operator name.
  4. Enter the Project name, if desired.
  5. Enter Sample pan position for calibration sample along with the sample size and pan mass.
  6. Click the Reference field to display the drop-down table and select the correct reference pan. If the correct pan and pan mass are not in the table, click Edit at the bottom of the table to enter applicable values.
  7. Enter any comments or Notes, if desired.
  8. Modify the method parameters.
  • Calibration must be performed using the same atmosphere and cooling accessory that will be used in subsequent experiments. Obviously, the lower and upper temperatures chosen for calibration must be within the operating range of the cooling accessory selected. In addition, the temperature range for calibration should be at least as broad as the range that will be used in subsequent experiments. When using Sapphire, the typical temperature ranges are: –90 to 400°C for the RCS90, –40 to 400°C for the RCS40, and 50 to 400°C for the FACS.
  • Enter the heating Ramp rate. Heat the material at the same heating rate that you will use in your subsequent experiments.
  • Enter the Start temperature and Final temperature.
  1. When sample and experimental input is complete, select from one of the following options:
  • Run All: Sends the calibration experiment to the Running Queue and execute run sequence.
  • Queue All: Sends the calibration experiment to the Running Queue for execution.
  • Schedule All: Schedules the calibration to run at a later specified date. Allows you to select how often the calibration should be run, the time, and the begin and end date.

Refer to DSC Calibration Overview for information on setting up auto calibrations and scheduling runs.

Standard Heat Capacity Calibration Setup Using the Template

This calibration requires one experiment using a heat capacity standard. A general procedure for calibration is given below. Experiment setup follows the same general procedure as a standard experiment (see Defining DSC Experimental Parameters).

  1. Prepare the sample and reference pans as described above in Sample Preparation.
  2. Access the Design View and add a run.
  3. Click on the associated Design Run Form (Single Run Form).
  4. Click the Sample bar. Enter the required test parameters for the experiment.
  • Note that the Name field varies based on the procedure mode/test type selected below. For heat capacity calibration,you must select the standard from the available list after selecting Calibration/Standard Heat Capacity as the procedure type.
  • Click the Procedure bar. Enter the required test parameters.
  • Select the Mode to Calibration.
  • Select the Test type as Standard Heat Capacity. The associated procedure template displays. Enter the required information.

  • Lower Temperature & Upper Temperature: Calibration should be performed using the same atmosphere and cooling accessory that will be used in subsequent experiments. The selected lower and upper temperatures for calibration must be within the operating range of the selected cooling accessory and the temperature range for calibration should be as broad as the range that will be used on your samples in subsequent experiments.
  • Ramp: Enter the heating rate. Heat the material, at the same heating rate that you will use in your subsequent experiments.
  • Once the experiment is setup, copy the experiment to the Running Queue and click Start to begin the calibration.
  • Once the experiment is complete, analyze the data.

Analyzing Standard Heat Capacity Calibration Data

Once the calibration experiment completes, analyze the data following the instructions below.

  1. Open the Heat Capacity calibration file in TRIOS.
  2. Select the file (or multiple files if an average of multiple runs is desired) in the File Manager Results pane.
  3. Select Calibrate > Heat Capacity from the Instrument menu options. Alternatively, you can select this option from the File Manager right-click pop-up menu or select the calibration run(s) in History and right-click, then select Calibrate > Heat Capacity.
  4. When you select Calibrate > Heat Capacity, a new document is created that contains the associated calibration results.

Several views are available within this document. 

  • Spreadsheet (the first Step tab in the above image): A table view of the associated calibration results
  • Graph (the second Step tab in the above image): A graphical view of the associated calibration results.
  • Results Summary (the Calibrations tab in the above image): A summary of the calibration parameters. Change the calibration name and add a description, if desired.
  1. On the Results view, select from one of the following options:
  • Save Results: Sends the results to the calibration database, but does not enable or apply the results.
  • Save and Apply: Sends the results to the calibration database and enables the calibration and then applies the results.
  • NOTE: This calibration document will automatically be saved to the following folder: C:\ProgramData\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows 7 or 8 or C:\Documents and Setings\All Users\TA Instruments\TRIOS\Discovery DSC\CalibrationDocs in Windows XP.

Selecting Standard Heat Capacity Calibration Results

The Discovery DSC stores all the calibration results in a database internal to the instrument. This allows you to access and set calibrations accordingly to meet the needs of your experiment. Selection of the calibration set is made on the Calibration Data page.

After analyzing and saving the results, you can apply the results so they are used in future experiments.

Access the Calibration > Calibration Datawindow from the File Manager to access the Calibration Data page.

Verify the desired calibration set is selected after applying the new calibration result.
