Although the experiment temperature is defined in the test procedure, it is often desirable to set the temperature and other environmental control options prior to starting the experiments. These parameters can be changed by accessing the General tab of the Control panel within TRIOS (screen shown below). The options in these panels vary based on selected cooler type.

Within these windows you can perform the following operations:

  • Monitor current temperature and purge gas status
  • Change the standby temperature set point
  • Enable/disable cooler unit
  • Change gas source and flow rate
  • Open and Close the AutoLid
  • Turn on/off the air cool control

Follow these instructions to use access this panel:

  1. Select Experiment tab > Controls > General tab on the Control panel.
  2. Use the controls on the panel to perform various functions as described below.
Discovery DSC General Control Panel Functions
Item Function
Lid Toggles the Lid open and closed.
Event Toggles the Event switch on and off. If used in conjunction with the RCS or LN2P, toggles the RCS or LN2P on and off.
Air Cool Toggles Air Cool control on and off.
DSC Options Opens the Discovery DSC options settings.
Used to view the status of the temperature systems and set the standby temperature.
Used to view the status of the environment system and set the gas selection and flow rate.

Changing the Standby Temperature Set Point

It may be desirable to alter the temperature set point outside of a test condition; for example, one may wish to change the current idle temperature. This can be accomplished by changing the temperature set point on the TRIOS Control panel.

  1. Open the TRIOS General Control panel.
  2. Expand the Temperature drop-down section.
  3. Enter the desired value in the field to the right of Standby, then click Standby.

Enabling/Disabling the Cooler Control

If your current cooler selection is the RCS and it is set to Event control, you can enable or disable the cooler using the General Control panel:

  1. Open the TRIOS General Control panel.
  2. The current state of the Event control is shown for RCS. Click RCS to switch the state of this control. Alternatively, toggle the Event control using the Event toolbar icon within this panel.

Switching the Gas Source and Flow Rate

The connected gas supply is set within the Discovery DSC General page within the TRIOS options. Through this control panel, you can switch between Gas 1 and Gas 2, as well as specify the desired flow rate. See below for information on changing these settings.

 

Changing Gas and Flow Rate Settings
Select the desired gas source (gas 1 or 2) from the drop-down selection.
Set the desired flow rate, and then click Flow rate.

Opening and Closing the Lid

There are various ways to open and close the DSC Cell Lid. Select from one of the options below:

  • From DSC keyboard on instrument.
  • From the instrument user interface, if applicable.
  • From DSC General Control panel within TRIOS software. Click the Lid icon to toggle between open and close state.

Turning On and Off Air Cool Control

Use the Air Cool icon within the General Control panel to open (on) or close (off) the internal solenoid valve connected to the Cooling Gas connector on the exterior of the instrument.

It is not recommended that you turn off Air Cool while running an experiment with a Finned Air Cooling System (FACS).
