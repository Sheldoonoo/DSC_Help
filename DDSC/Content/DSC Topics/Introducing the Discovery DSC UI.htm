Overview

The Discovery DSC includes a keypad and an integrated user interface display for local operator control. The functions of the user interface change depending upon the view displayed. This section briefly describes the basic layout of these functions.

The instrument user interface has four basic sections:

  • The status line along the top of the display indicates the current instrument status, run selection, and sample temperature.
  • The primary function keys allow for easy access of common functions.
  • Two customizable view panels allow for real-time instrument status and associated actions.

Use the following keys for the main functions of the instrument.

 

Primary Function Keys on User Interface
Button Description
Start Begins the experiment. This is the same function as Start on the instrument control software. Start automatically loads the selected sample pan and closes the furnace, if necessary, before beginning the experiment.
Stop If an experiment is running, this key ends the method normally, as though it had run to completion; i.e., the method-end conditions go into effect and the data that has been generated is saved. This is the same function as Stopon the instrument control software. If an experiment is not running (the instrument is in a standby or method-end state), the Stopkey will halt any activity (air cool, all mechanical motion, etc.). If an Autosampler sequence is in progress, Stop halts the sequence.
Lid Open/Close Toggles between opening and closing the AutoLid on the DSC cell.
Load/Unload Displays the Autosampler control view, which allows loading or unloading from/to the selected pan position.
More... Displays a menu of other instrument control options such as screen calibration, shut down, and restart instrument.

View Panels

The user interface allows for the display of two simultaneous view panels, each providing real-time instrument information. These panels are selected through the Set View button. These panels also include additional functionality pertinent to the selected view. In cases where user input is required, the view switches to full-screen mode until the input is satisfied. A list of available views is described below.

  • Run Details
  • Signals
  • Autosampler
  • Notifications
  • Instrument Properties

 

Run Details

Provides a summary of the sample and method information for the current run. Select between these two tabs to display the associated information. When the instrument is idle, access to the Quick Run setup is available through this panel.

Signals

Displays the real-time signal data generated directly from the instrument. Up to six signals can be displayed.

To change the signal being displayed, press , then select the desired signal from the drop-down list.

Back to top

 

Autosampler

Displays a graphical representation of the Autosampler tray, indicating the status of the active sequence (that is, which pan positions have been configured in the active sequence and which runs are complete, active, or pending). Press the tray image to access further Autosampler operations such as pan load/unload operations and run details for running queue.

In addition, this panel provides access to other Autosampler operations.

  • Calibrate: Select to access Autosampler calibration functions.
  • Reset: Select to reset the Autosampler.
  • Park: Select to reposition the Autosampler in the park position.

Back to top

 

Full View

Displays a graphical representation of the Autosampler tray and cell, indicating the status of the active run and sequence (e.g., which pans are loaded in the cell and which pan positions have been configured in the active sequence). By pressing on a pan number, you can view the run details associated with that run.

In addition, use this view to manual load and unload sample pans.

  • To load, press the ← button then select the desired pan position.
  • To unload, press the → button then select the desired pan position.

To return back to the Autosampler summary view, press < Back.

Back to top

 

Notifications

Displays a log of the most recent instrument messages. The messages are in the format: Message #, Date, Time, and Description.

Press Clear to clear the messages from this view.

Back to top

 

Instrument Properties

Displays instrument information such as the software version, options, and IP configuration.

Select Network to change the network configuration settings.

The Name and Location fields may be customized. Selecting one of these fields automatically displayed the keyboard. Once complete, select the X button to accept the changes.
