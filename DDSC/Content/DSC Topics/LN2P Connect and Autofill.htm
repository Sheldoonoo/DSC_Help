Overview

  • CAUTION: Do not use other lines to fill the LN2P, as lower flow resistance will increase Dewar pressure.

To use the local filling capability, connect the cooling accessory to a low pressure bulk source of liquid nitrogen, then fill the Dewar.

There are several ways to autofill the Dewar. These autofill methods are described in detail in the sections below.

  • Using the Common Cabinet User Interface, if available. Select LN2P Controls from the Set View options list, then press Fill Tank.
  • Using TRIOS software :
  • Within the General Control panel, click on the Toggle Fill icon to fill the Dewar at any time. Note that LNP Cooler needs to be selected in order for this icon to be available. Refer to Setting Up DSC Cooler Options for more information.
  • Within the Options > Discovery DSC > Cooler options, enable the autofill function and set a % level; the Dewar automatically fills whenever the liquid nitrogen level reaches the user-set % level (regardless of whether or not a run is in progress). Refer to Setting Up DSC Cooler Options for more information.
  • Within the procedure, add one of two Fill Cooler segments. One Fill segment waits for the fill process to complete before moving on to the next method segment, while the other initiates a fill and moves on to the next segment while still filling the Dewar.

Connecting the LN2P to a Low-Pressure Bulk Source

Follow the instructions below to connect the LN2P to a source of liquid nitrogen:

  1. Arrange the low pressure liquid nitrogen (172 kPa [25 psig] max) bulk storage source physically close enough (within 1.8 m [6 ft]) to the LN2P so that the autofill transfer tube can be easily connected between the source and the LN2P. Likewise, the LN2P and the instrument need to be in close proximity to allow connection of the 1.8-m (6-ft) coolant delivery line.

  2. Connect the LN2P for automatic filling as follows:

  3. Attach the transfer tube to the LN2 FILL VALVE located on the motor housing (shown in the figure below).

  1. Attach the other end of the transfer tube to the bulk storage container using the union and adapter fitting (provided in the accessory kit).
  2. Install the cooling head on the DSC as directed in the Liquid Nitrogen Pump (LN2P) Getting Started Guide if it is not already installed. The cooling head must be in place on the cell and the cell must be operational before filling the LN2P.
  3. Plug the LN2P power cord into an outlet, and connect the USB cable between the instrument and the LN2P.
  4. Follow the directions in the sections below to fill the Dewar with liquid nitrogen before beginning an experiment.

Initial Autofilling

The LN2P must be filled before cooling experiments can be performed on the Discovery DSC. Follow the instructions in this section to fill the LN2P.

 

Autofilling using TRIOS Software

  1. Using TRIOS software, connect to the desired instrument.
  2. From the TRIOS Control panel, select General > LN2P > Toggle Fill . The LN2P fills automatically.
  3. The autofill shuts off when the Dewar is full or the bulk storage tank is empty. Once the Dewar has been filled initially, you can set up programmed autofilling as described in Programmed Autofilling below.


Autofilling using the Common Cabinet User Interface

  1. From the Common Cabinet User Interface, select LN2P Controls from the Set View options list.
  2. Press Fill Tank.

  1. The autofill shuts off when the Dewar is full or the bulk storage tank is empty.

Programmed Autofilling

Autofilling also refers to the automatic refilling of the LN2P from the bulk storage tank between runs. This section explains how to set up the LN2P and the connected instrument to allow autofilling. Refer to Setting Up DSC Cooler Options for additional information.

The DSC automatically controls the LN2P, which supplies liquid nitrogen to the cooling head.

To automatically refill the LN2P with liquid nitrogen at any point:

  1. In TRIOS software, access TRIOS Options > Discovery DSC > Cooler page.
  2. Select Autofill LNP if below, then enter the desired liquid level percentage.
  3. NOTE: The LN2P is normally filled from a bulk tank located near the unit. If you need to fill your LN2P Dewar with liquid nitrogen from a remote source (i.e., source not located near your unit), refer to the Liquid Nitrogen Pump (LN2P) Getting Started Guide for instructions on remote filling of the Dewar.

Fill Method Segment

There may be instances in which it is optimal or necessary to fill the LN2P between runs during an experiment. For example, this would be useful if, prior to starting a long experiment, you know that the LN2P will run out of liquid nitrogen before a particular run completes. In such cases, use the Fill Cooler method segment to fill the LN2P at a point of your choosing.

Refer to Creating Custom Methods and the Available Method Segment list for information on using the Fill Cooler method segments.
