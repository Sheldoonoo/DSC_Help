Discovery DSC: Loading and Unloading Pans in the Autosampler

In this topic
Loading and Unloading Pans using the User Interface
Loading and Unloading Pans using TRIOS Software

As part of an Autosampler sequence, the specified sample and reference pans automatically load into the cell, followed by automatic closure of the lid. Once the experiment is completed, the lid opens, the pan(s) are removed, and the next sample pan loads. This process repeats until the end of the run sequence.

It is also possible to manually instruct the instrument to load and unload specified pans. Refer to the sections below.

Loading and Unloading Pans using the User Interface

To manually load and unload pans into the cell using the instrument user interface, follow these instructions:

  1. Select Set View > Autosampler, then select the image of the Autosampler sample pan tray.
  2. The AutoLid opens automatically. If it is in the closed position, select Lid to open it.
  3. To load, select ←, then select the desired pan position.
  4. To unload, click →, then select the desired pan position.

Loading and Unloading Pans using TRIOS Software

To manually load and unload pans into the cell using TRIOS software, follow these instructions:

  1. In TRIOS software, click Control panel > Auto Sampler. The following Control panel screen displays.

  1. To load a sample pan:
  2. Enter the sample pan Autosampler position number in the field to the right of Load sample pan #.
  3. To load, click Load sample pan #. To unload the specified sample pan, click Unload Sample .
  4. To load a reference pan:
  5. Select the reference pan Autosampler position number from the drop-down menu to the right of Load reference pan #.
  6. To load, click Load reference pan #. To unload the specified reference pan, click Unload Reference .
