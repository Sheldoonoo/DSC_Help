  1. Click the TRIOS button at the top-left corner of the application.
  2. Click Options from the drop-down menu.
  3. Proceed to the applicable section below for configuration details.

Viewing/Editing Instrument Network and Properties Information

To configure the instrument network and properties options:

  1. Open the Options screen.
  2. Select Discovery DSC > Information.

 

This page provides a summary of the network and properties information for the instrument. Use this window to configure the following options:

  • Instrument Information: This section displays the current network configuration (IP Type, IP Address, etc.). Changes to this configuration can be made through the instrument user interface.
  • Instrument Name and Location: These parameters are customizable fields that allow you to differentiate instruments by name and/or location. Enter the desired information. When finish, click OK to save. Alternatively, you can set this information from the instrument user interface, if applicable.
  • Instrument Software: Displays the current software versions of each of the instrument components along with their serial number. To take advantage of new features in the TRIOS Software, it may be necessary to update the instrument firmware. Click Update to update this software. Refer to Updating Instrument Software (Firmware) using TRIOS Software for instructions.

Setting Up Global DSC Options

To configure the instrument stage control options:

  1. Open the Options screen.
  2. Select Discovery DSC > General to display options that allow you to modify certain parameters used by the instrument while it performs experiments.

  • Transition Direction: Select the method in which the data is plotted (Exotherm Up or Exotherm Down). See the figures below for examples.

  • Heat Flow Selection: Choose the desired heat flow signal from the drop-down list. More information on T1, T4, and T4P can be found in the help topic About Tzero DSC.
  • Data Sampling Interval: Enter the desired Data Sampling Interval in seconds per point. This instrument parameter defines how often the system reads and stores a data point. For example, if 2.0 is entered, the system stores a data point every two seconds. Range is between 0.1 and 1000.0. The default value is 0.2 seconds/point for DSC and MDSC modes.
  • Lid Type: Use to specify the currently-installed lid type to limit the allowable upper temperature. Choose between:
  • Standard: For temperatures below 400°C . Note that the standard lid can be recognized by a "c" that is engraved on the lid.
  • High Temperature: For temperatures up to 725°C
  • Gas 1 / 2: Use Gas 1/2 to specify the purge gases that are connected to the corresponding ports on the instrument. A purge gas is a gas, often an inert gas such as nitrogen, that flows through the instrument during the experiment. The gas flow is set within the General panel.
  • Stop experiment when flow rate deviates from the set value: In addition, you can check this option, if you would like to have the experiment stop in the event the flow rate cannot be maintained during an experiment (e.g., if the purge gas tank becomes empty).
  • Instrument Database Backup: TRIOS includes an automated mechanism to backup important records stored on the instrument in the event of a hardware failure so the information can be restored. These functions are meant for use by TA Service Representative; they are setup by default and require no user intervention.

  • NOTE: If you plan to connect to your instrument from more than one computer, you may wish to turn off backups on the secondary computer by enabling the Prohibit any backup on this computer option.

Setting Up DSC Cooler Options

To configure the instrument Cooler options:

  1. Open the Options screen.
  2. Select Discovery DSC > Cooler.

  • Cooler Selection: Choose the type of cooling device that is to be used with your DSC from the drop-down list.
  • NOTE: When using the FACS with Quench Cooler for FACS, ensure that the Cooler Selection identified is the FACS and not the Quench Cooler. If the wrong cooler is identified, your results will be inaccurate.
  • Secondary Purge Activation: When using the RCS or LN2P, it is recommended that a source of dry nitrogen gas be connected to the cooling gas port if the instrument is used in high humidity environments or when recovering from moisture contamination. When the Activate secondary purge when lid is opened checkbox is used, the secondary purge will supply additional dry gas into the RCS and LN2P housing when the AutoLid is opened. This option should be left blank when there is not a secondary gas connected to the cooling gas port and/or the port is capped.
  • Leave Cooler On: Check this option to leave the RCS or LN2P on between runs. By default, this option is checked.
  • Autofill LN2P if below: Enter a percentage to automatically fill the LN2P, if desired. This setting indicates that you want the LN2P to automatically fill when the level of liquid nitrogen falls below the specified percent. The fill process, when activated, will fill to completion. If left unchecked, you will need to manually fill the LN2P when needed.

Setting Up DSC Autosampler Options

To configure the instrument Autosampler options:

  1. Open the Options screen.
  2. Select Discovery DSC > Auto Sampler.

The stop options described below instruct the Autosampler on the action to take when the run sequence has been stopped or runs to completion. These options will override the Post-Test conditions for the last run in the sequence. Select the desired stop sequence option.

  • Unload Pans: Check this option for the Autosampler to automatically unload the pans from the cell and place them either in the sample tray or the waste bin. If this is left unchecked, the pans will not be unloaded until you perform that step yourself.
  • Leave Cooler On: Check or uncheck this box to control the state of an external device (RCS or LN2P) connected to the instrument with the event cable at the end of the Autosampler run sequence.
  • Check the box to turn the event device on at the end of the last Autosampler run in the sequence. If the RCS or LN2P was on at the end of the run sequence, it will remain on if this box is checked.
  • Uncheck the box to turn the event device off at the end of the run sequence.
  • NOTE: In the event that the run sequence is stopped due to an error, the RCS or LN2P will automatically be turned off to prevent frosting, regardless of your selection.
  • Enable Autosampler and Autolid operation: Use this option when you are operating a Photocalorimeter Accessory (PCA) with the DSC instrument. For proper operation of the instrument, the Autosampler and Autolid are disabled as a safety when the PCA is installed. To reactivate the Autosampler and Autolid after removing the PCA, check this option.
