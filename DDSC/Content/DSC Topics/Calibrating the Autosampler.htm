Performing a Verification and Adjustment Procedure

In the event that the Autosampler drops a pan or you receive an Autosampler error, it may necessary to refine your Autosampler calibration. This is accomplished using the Verification and Adjustment Procedure, the most effective Autosampler calibration. This routine guides the operator through verification and adjustment of sample tray positions 1, 26,and R1,as well as cell sample and reference positions. In most cases, no other adjustments or calibrations are necessary.

To begin verification and adjustment routine, follow these steps:

  1. From the Discovery DSC user interface, access the Autosampler calibration functions by selecting Set View > Autosampler > Calibrate.

  1. Remove any pans from the following positions: sample tray positions 1, 26, and R1, and any pans from the cell.
  2. Place a clear sapphire disk in sample tray position 1. When ready to begin calibration, select Start.
  3. The Autosampler moves to the specified location for verification. If the position is acceptable, select Yes to continue. Otherwise, select Adjustto adjust.
  • If you select Yes, the sapphire disk moves to the next position in the verification sequence.
  • If you select Adjust, the adjustment window displays, allowing for manual adjustment (see Using the Autosampler Adjustment Window for further details).Once you are satisfied with the adjustments, select Saveand press < Back to return to step 4.
  1. The process repeats until all the positions are verified.

Additional Procedures

Other calibration functions are available within Options >, although rarely will any of these calibrations be necessary. Click a link below to open your desired Help topic.

  • Full Auto Calibration: Used to perform tray and cell calibration when first calibrating the Autosampler
  • Cycle Test: Used to cycle the Autosampler motors
  • Motor Test: Used to manually test individual motors
  • Lid Calibration: Used by service to adjust the lid calibration
  • Other Calibrations: Allows access to the full range of calibration positions

Using the Autosampler Adjustment Window

The Autosampler adjustment screen allows you to make adjustments to the arm rotationally and laterally. Adjustments from this screen are applied to selected position only.

  • NOTE: The tray height cannot be calibrated from this screen; it is calibrated either during the Full Auto Calibration or the Tray Height calibration procedure.

  1. Use the directional arrows to move the arm to the desired position, as well as the following:
  • Use the Close Grip and Open Grip functions to grip and release the pan.
  • Change the increment in which the Autosampler adjusts by selecting Course,Medium, or Fine. Press this button to toggle the available adjustment settings.
  • The Current Offsets for each position are shown on the screen. The offsets displayed are based on the change from the last saved position (0=no change).
  • NOTE: The image varies based on calibration position.
  1. Once the position is refined, select Save.
  2. Select Test to verify the position.
  3. If desired, select either of the following:
  • Details: Displays the offset values for the selected calibration, and displays Restore options (Last Saved and Factory Defaults).
  • Reset: Use to reset the Autosampler arm to the "park" position.
  1. Once satisfied with the calibration, select < Back to return to the calibration selection window.
