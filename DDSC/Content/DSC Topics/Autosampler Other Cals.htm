Discovery DSC: Additional Autosampler Calibrations

The DSC Autosampler components can be calibrated individually using the Other Calibrations menu, available through the user interface. Not all of the directional buttons will be applicable and therefore, not available, depending upon the option selected.

To access Other Calibrations:

  1. From the Discovery DSC user interface, access the Autosampler calibration functions by selecting Set View > Autosampler > Calibrate.
  2. Select Options > and press Other Calibrations. The following screen appears.

  1. Select the desired function and press Go.
  2. Refer to the table below for information regarding the individual calibrations.
Calibrations
Function

Description

Tray Positions 1, 26, and R1 These functions are used to set the specified tray positions.
Cell Sample and Reference Positions These functions are used to set the specified cell positions.
Cell Calibration Start

This function is used to set the cell calibration starting position. The starting position will then be used when auto cell calibration is performed.

  1. Remove any pans left in the cell. Place the calibration fixture in position over the sample and reference positions.
  2. Select this position and select Go.
  3. Use the Down/Up and Left/Right arrows to position the grippers so that all three gripper fingers are above the calibration fixture, as shown below.
Tray Calibration Start

This function is used to set the tray calibration starting position. The starting position will then be used when auto tray calibration is performed.

  1. Grasp the center of the sample tray and pull it straight up to remove the tray, exposing the plate underneath.
  2. Select this position and select Go.
  3. Use the Down/Up and Left/Right arrows to position the grippers so that they are located in the largest calibration disk (circled in the figure below)
Grip Open

This function is used to set the gripper open position.

  1. Remove any pan from sample tray position 1.
  2. Select position and select Go.
  3. Select Goto Tray 1 to move the grippers to tray position 1.
  4. Adjust the grip open magnitude by using the Grip Open and Grip Close buttons.
  5. Once satisfied, select Save.
  6. Test this function using a sample pan and using the Grip to test the calibration.
Grip Close

This function is used to set the gripper close position.

  1. Remove any pan from sample tray position 1.
  2. Select position and select Go.
  3. Select Goto Tray 1 to move the grippers to tray position 1.
  4. Select Grip.
  5. Adjust the grip open magnitude by using the Grip Open and Grip Close buttons.
  6. Once satisfied, select Save.
  7. Test this function using a sample pan and select Grip.
Park Position This function is used to set the Autosampler park position.
Ship Position This function is used to set the Autosampler shipping position.
Waste Bin Position Select this function to make adjustments to the Autosampler's positioning when delivering a pan to the waste bin.
Loader Position This function is used to set the loader position. The loader position is used to facilitate loading pans into the sample tray while an autosampler sequence is active.
Tray Height This function sets the range of motion for the Autosampler arm in relation to the tray height.
Cell Height This function sets the range of motion for the AutoLid arm in relation to the cell height.
