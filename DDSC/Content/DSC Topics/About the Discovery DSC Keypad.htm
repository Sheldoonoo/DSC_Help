About the Discovery DSC Keypad

The Discovery DSC includes a keypad designed for local control of common instrument functions.

Experiment information and instrument parameters are entered using TRIOS software or the user interface.

 

Key Functions
Key Name Description

Start

This key begins the experiment. This is the same function as Start on the instrument control software.

This button is white when the system is in standby mode, yellow when a method is loaded (at least one run in the TRIOS Running Queue), and green when the system is running.

Stop

If an experiment is running, this key ends the method normally, as though it had run to completion; i.e. the post test conditions go into effect and the data that has been generated is saved. This is the same function as Stop on the instrument control software. Press this key twice to stop temperature control.

This button is always red.
Open/Close

Toggles between the opening and closing of the Discovery DSC AutoLid.

This button is always white.
Load/Unload

Removes the pan from the sample cell position and returns it to its original location (if applicable), and loads the pan from the load position into the sample position. Does not load samples from the tray.

This button is always white.
Reset

Resets the Autosampler.

Caution: This command erases all current statuses for the sample and/or reference pan positional information.

This button is always white.
Standby

Activates the pre-programmed standby temperature when the system is in idle mode.

This button is green when the system is in standby mode, green when a method is loaded (at least one run in the TRIOS Running Queue), and yellow when the system is running.
