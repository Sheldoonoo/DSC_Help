Defining a Data Range

In this topic
Hide Function
Reduce Function

There are two options that one can use to define a data range: Hide or Reduce. See below for more information.

Hide Function

Use the Hide data functions on the Edit tab to narrow the data range for the selected data. Unlike rescaling, the selected data is permanently “hidden.” Selecting Unhide All re-displays the hidden data.

To use function, follow the instructions below:

  1. Click on the desired curve.
  2. Using the mouse, highlight the data of interested by holding down the mouse button until the range is selected.
  3. From the Graph > Edit tab or the right-click graph pop-up menu, select from one of the following options:
  • Hide selected: Hides the selected data, rescaling the axis if appropriate
  • Hide selected – all variables: Hides the data outside of the selected data region of data for all axes, rescaling the axis if appropriate
  • Hide unselected: Hides the data outside of the region of the selected data, rescaling the axis if appropriate
  • Hide unselected – all variables: Hides the data for all axes outside of the region of the selected data, rescaling the axis if appropriate
  • Hide selected - all steps: Hides the data outside of the region of the selected data for all steps, rescaling the axis if appropriate
  • Hide selected - all steps, all variables: Hides the selected region of data for all steps and axes, rescaling the axis if appropriate
  • Hide unselected - all steps: Hides the data outside of the region of the selected data of the selected curve, rescaling the axis if appropriate
  • Hide unselected - all steps, all variables: Hides the data outside of the region of the selected data for all axes and steps, rescaling the axis if appropriate
  • Hide range: Displays a window to manually enter in the desired data range. To display only a selected range of data, enter the desired range and click Display. All curves on the graph will be modified. To display only a selected range of data for a single curve, first highlight the portion of the curve you want displayed. Range values automatically populate the range fields in the Hide Range dialog box. Select Single step to display only the curve that is highlighted, or leave it unchecked to modify a single curve only, leaving the rest of the graph untouched.

  • Unhide All: Re-displays all hidden data.
  1. NOTE: If there is more than one step displayed and Single Step option is selected, the selected action will apply to all steps.

Alternatively, you can use a Zoom Box to specify the data region. This is a particularly useful method for overlay plots and situations where the desired data is difficult to select:

  1. Position the cursor in the general area you wish to mark.
  2. Hold down the left mouse button and drag to create a box. [Draw a box encompassing the desired data region.] Release the mouse when the desired data is selected.
  3. Move the cursor inside the box and right-click to display the pop-up menu.
  4. Select the desired Hide function as described above.

Reduce Functions

Use the Reduce data functions to narrow the data range for the selected data. Unlike rescaling or hiding, the selected data is removed from the file. However, selecting Reload Data from the spreadsheet reloads the original data (including restoring the removed data).

To use the Reduce function, follow the instructions below:

  1. Verify your selection of the desired reduction technique from the TRIOS Options > Analysis window.
  2. Click on the desired curve.
  3. From the Graph > Edit tab > Data toolbar, select Reduce. A window appears requesting the number of points to Reduce to.

  1. Enter the number of points to reduce to.
  2. If Single step is selected, then any Hide command will apply just to the selected curve. If not selected, then any Hide command will apply to all curves.
  3. Click OK when finished.

To use the Remove and Retain functions, follow the instructions below:

  1. Click on the desired curve.
  2. From the Graph > Edit tab > Data toolbar, select Remove or Retain as appropriate. Remove discards the range selected and keeps all other data; Retain keeps the range selected and discards all other data. A window appears displaying the x-range selected:

  1. The x-range may be adjusted by entering new values, if required. Additionally, the x-range lower or upper limits may be disabled completely by unchecking the check boxes, allowing ranges such as >a or <b to be specified.
  2. Select either Remove specified range or Retain specified range from the Type drop-down menu.

  3. If Single step is selected, then any Hide command will apply just to the selected curve. If not selected, then any Hide command will apply to all curves.

  4. Click OK when finished.
