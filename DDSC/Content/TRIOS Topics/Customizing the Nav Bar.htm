About the Navigation Pane/Bar

The File Manager and Control panel both use a navigation bar to display and customize their panels. The view buttons and commands on the navigation bar correspond to the different views on the navigation pane, such as Last point, Gap, Auto Sampler, and Motor (in the case of the Control panel), and Experiments, Results, Calibration, and Geometry (in the case of the File Manager). Depending on the chosen view, a different set of panes, folders, and information displays. For example, you can display the view buttons in a small or large size, and you can add or remove buttons when needed.

Adding or Removing Buttons in the Navigation Pane

File Manager

There are multiple ways to add or remove buttons in the File Manager navigation pane. On the navigation bar, click Configure Buttons . Then use either of the following methods:

  • Scroll over Add or Remove Buttons to display the pop-up menu, and click your desired button to add or remove the selection from the navigation pane.
  • Click Navigation Bar Options..., check or uncheck the display button to show or hide your selection, then click OK.
  • NOTE: To restore the default button arrangement, click Reset.

Control Panel

To add or remove buttons in the Control panel navigation pane:

  1. On the navigation bar, click Configure Buttons .
  2. Click Navigation Bar Options.

  1. Check or uncheck the display button to show or hide your selection.
  2. Click OK.
  3. NOTE: To restore the default button arrangement, click Reset.

Resizing the Pane

To increase space in the Control panel or File Manager pane quickly, you can change from large buttons that are displayed vertically on the navigation pane to small buttons that are displayed horizontally across the navigation bar.

To resize the pane vertically:

  1. Place the mouse cursor over a horizontal splitter bar. Splitter bar locations are circled below (Control panel on left, File Manager on right).

  1. Drag the horizontal splitter bar up or down to resize the pane. As you drag the horizontal splitter bar towards the bottom of the navigation pane, the large view buttons disappear and are converted to small icons that appear in a single row on the navigation bar.
  2. NOTE: To resize the pane horizontally, place the mouse cursor over one of the pane's vertical borders and drag the border to the left or the right.

Changing the View Button Display Order

To change the order in which the view buttons display on the navigation pane:

  1. On the navigation bar, click Configure Buttons .
  2. Click Navigation Bar Options.

  1. To rearrange the button order, in the Display buttons in this order list, select the button that you wish to change, and then click Move Up or Move Down as applicable.
  2. NOTE: To restore the default button arrangement, click Reset.

Alternating Between Large and Small View Buttons

To change a large view button on the navigation pane to a small icon on the navigation bar (and vice versa):

On the navigation bar, click Configure Buttons .

  • Click Show More Buttons to add the next icon on the navigation bar to the list of view buttons on the navigation pane.
  • Click Show Fewer Buttons to remove the next view button on the navigation pane and add it (in icon form) to the navigation bar.

To quickly see more buttons or folders, use the horizontal splitter bar to resize the navigation pane. See Resizing the Pane for more information.
