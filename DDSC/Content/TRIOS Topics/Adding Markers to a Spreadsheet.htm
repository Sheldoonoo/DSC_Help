TRIOS software offers the ability to add markers to rows and columns in a spreadsheet. Use the functions within the Markers toolbar (shown in the figure below) to add textural comments, set bookmarks, or mark any invalid data.

Highlighting a row or column will enable the available functions in the Markers toolbar. Numbered rows allow use of all functions in the Markers toolbar, while column headings allow for adding comments only.

See the headings below for detailed information related to these functions. The instructions below assume an open data file with the appropriate spreadsheet tab selected.

Adding New Comments

To add a new comment to a row or column: 

  • Click on the column header or row number to select it.
  • Select Format tab > Markers toolbar > Add Comment , or right-click and select Add Comment from the pop-up menu. The Add comment window appears.
  • Enter the desired text.
  • Click OK when finished.
  • NOTE: See Editing/Deleting Comments for information on adding additional comments to an existing comment.

Editing/Deleting Comments

 To delete or add to an existing comment:

  • Position your cursor over the comment marker and right-click to display the pop-up menu.
  • Select Edit Comments. The Edit comment window appears, displaying the comment text and author name.
  • Perform one of the following functions:
  1. Add additional comments: Click Add, then type your comment at the cursor.
  2. Delete a comment: Click on the comment to select it, then click Delete.
  • Click OK when finished.

Marking Data Invalid

You can select one or more rows to mark the data as invalid. The Mark Invalid function is not available for columns of data.

  • Marking data as invalid will cause the marked data to be ignored on the Graph view, if that option was selected in the Properties panel.

To add an invalid data marker(s):

  • Select a row by clicking on the row number. To select multiple rows, click and hold the left mouse button on the first row you wish to mark and drag the mouse pointer down to the last row you wish to mark.
  • Select Format tab > Markers toolbar > Mark Invalid , or right-click and select Mark Invalid from the pop-up menu.

Setting Bookmarks

Bookmarks can be used to mark a row position for future recall. The Set Bookmark function is not available for columns of data.To add a bookmark to a row:

  • Select a row by clicking on the row number. To select multiple rows, click and hold the left mouse button on the first row you wish to mark and drag the mouse pointer down to the last row you wish to mark.
  • Select Format tab > Markers toolbar > Set Bookmark , or right-click and select Bookmark > Set Bookmark from the pop-up menu.

Clearing Markers

Use the Markers drop-down menu to clear various types of markers from the spreadsheet. See below for additional information.

To Clear a Marker

  • Select the row or heading that contains the marker you wish to remove.
  • Select Format tab > Markers toolbar > Markers drop-down menu, and select the applicable item:
  • Clear Bookmark or Clear Invalid Data Marker removes the applicable item from the row or column.
  • Clear All Bookmarks or Clear All Invalid Data Markers removes all applicable items from the spreadsheet.

Navigating Between Spreadsheet Markers

Go To Next and Go To Previous are found in the Markers toolbar on the Formatting tab. This function navigates between markers on the spreadsheet. Select the desired marker type from the drop-down list. The program searches above or below the current row position for the selected marker and moves to that position.
