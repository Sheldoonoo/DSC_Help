Linear Transform

The Linear Transform feature allows you to transform your graph data using a linear equation. Follow the steps below to perform a linear transform.

  1. Right-click on the graph and hover on Linear Transform. Select which variable to modify, Y Axis or X Axis.

  1. The Linear Transform dialog box for the selected axis displays.

  1. Enter the values for the equation and then click OK.
  • NOTE: 0 (zero) cannot be used as the multiplier or denominator.
