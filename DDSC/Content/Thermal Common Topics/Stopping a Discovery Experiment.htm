When an experiment is in progress, you can stop the experiment and/or sequence.

Stopping an Experiment

There are several ways to halt an experiment and save the data:

  • From the Experiment tab (Discovery TGA shown):

  • From the instrument user interface (select the Stop button on the instrument user interface, or press the Stop button on the instrument keypad):

After pressing Stop, the End of Test conditions activate. The end of test temperature conditions are defined within the Advanced setting of the experiment. The other end of test/sequence conditions are defined through the Discovery Options. For more information on the Discovery Options, see Configuring Instrument Options.

Stopping the Autosampler Sequence

When an Autosampler sequence is in progress, the following stop options are available through TRIOS software only, and not through the user interface or keypad.

  • Stop Immediately: Select this Autosampler stop option to immediately halt the active run or action (e.g., air cool) and terminate the sequence.
  • Stop After Current Run: Select this Autosampler stop option to halt the sequence upon the conclusion of the current run.

Select Go to Next to immediately halt the current run and continue to the next run in the sequence.
