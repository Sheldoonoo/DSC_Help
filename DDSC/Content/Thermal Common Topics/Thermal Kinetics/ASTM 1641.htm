Overview

ASTM Procedure E1641 is based on a kinetics approach originally developed by Flynn and Wall [1], which requires three or more determinations at different linear heating rates, usually between 0.5 and 50°C/minute. This approach assumes the basic Arrhenius equation:

Using a point of equivalent weight loss (decomposition) which is beyond any initial weight loss due to volatiles evolution, a plot of ln b vs 1/T is constructed. The slope of this straight line plot is then used to calculate activation energy (Eα), and the pre-exponential factor (Z) is calculated by the interactive method of Zsako and Zsako [3].

Experimental

In a preliminary TGA experiment, the substance of interest is measured at several different rates through its decomposition region. The resultant weight loss curves are then converted to conversion plots using the TGA Conversion function, and the data are collected into a single set of data at a specific conversion using the TGA Conversion set function, using a minimum of three different heating rates. This data can then be analyzed using the Flynn Wall Isoconversion model.
