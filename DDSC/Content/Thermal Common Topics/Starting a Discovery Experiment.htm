Discovery Instrument: Starting an Experiment and/or Run Sequence

Once your sequence (s) are set up and the associated sample(s) positioned on the Autosampler tray, you are ready to start the sequence. Runs are only executed from the Running Queue list, so within the File Manager - Experiments panel, move the configured runs from the Design View to the Running Queue for execution from within the File Manager Experiment panel by dragging the runs from the Design View to the Running Queue.

To start the sequence, select Start from one of the following options:

Within TRIOS software:

From the instrument user interface (select the Start button on the instrument user interface, or press the Start button on the instrument keypad):

  • NOTE: The Start button is only illuminated when the instrument is in an "idle" state and the Experiment view is displayed.
