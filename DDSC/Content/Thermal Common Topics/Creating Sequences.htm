Overview

Run sequences can be created in either the Design View or Running Queue sequence views. Both of these views contain the same basic tools for creating, modifying, and saving sequences.

The following are the basic steps to creating a run sequence using the Single Run form:

  1. Create a run sequence.
  2. Enter the experiment parameters (Sample and Procedure information). Experimental information can be entered from either the Single Run form or the Table View. Refer to the appropriate topic below:
  • Defining Discovery DSC Experimental Parameters
  • Defining Discovery TGA Experimental Parameters
  1. Append or Insert a new run to the sequence, then configure the run parameters accordingly.
  2. Repeat steps 2 and 3 until the desired number of runs are configured.
  3. Review the run sequence information in the Table View.
  4. If created in the Design View, copy the sequence to the Running Queue: Right-click the Run and select Copy to the Running Queue from the menu.

  1. Start the run sequence.

Creating Run Sequences

Access either the Design View or Running Queue from the File Manager Experiment panel. Both views contain tools to create, modify, and save sequences. These tools can be used to create sequences in a variety of ways, but the most direct method is described below. Note that as you become familiar with setting up experiments, you will find the Table View very useful in facilitating sequence creation.

To create a new sequence, do one of the following:

 

In Running Queue

  1. If no runs are currently set up, Empty appears within the File Manager list (as shown below). Click Empty to create a new run. If multiple runs (a sequence) already appear in the list, you can append a new run to the sequence.

Alternatively, create a new sequence by clicking Reset Sequence . Select this function and the Rest Sequence Options dialog box displays:

Select the desired options and click Continue.

  1. Proceed to the next section for information on organizing runs within the File Manager Experiments panel.

In Design View

  1. Expand the Design View (if not open) by clicking on the arrow in the right corner of the title bar.
  2. Click on the New Design View button to create a new sequence.

  1. Enter in some descriptive information, if desired, then click the Save button.

  1. An empty run displays in the Design View. Click Empty to create a new run (Run 1).

Working with Sequences in the File Manager – Experiments Panel

Run lists are available for Design View, Running Queue, and Incomplete sequence views. Depending on the view, the list of available functions varies. Functions are available from the toolbar and/or the right-click pop-up menu that allows for sequence creation and modification. See below for information on various functions available from the run list.

 

Run Functions
Function Description
Start Sequence Starts the run sequence within the Running Queue only.
Reset Sequence Opens a window that provides new sequence options.
Open Sequence File Click to load a saved sequence file, which replaces the current one.
Save Sequence to File Click to save the sequence to file for future recall.
Append Run Adds a copy of the selected run to the end of the sequence.
Insert Run Adds a copy of the selected run in the selected location.
Delete Run Removes (deletes) the selected run from the sequence.
Move Run Up/Down Used to move the selected run up or down in the sequence run order.
Filter Runs Display filter fields that allow you to quick find a run within the displayed sequence (applicable to Table View only).
Reordering Runs Drag and drop runs within the run list to reorder the list.

Working with Multiple Sequences in Design View

The Design View allows you to configure one or more Sequences while connected or offline from the instrument. To access the pop-up menu shown below, right-click the red dot that appears below the Design View Sequence list (within the File Manager).

See the descriptions below for information regarding the functions available in this view.

  • Sequence Information: Displays the Sequence information block that allows the user to enter information about the design sequence.
  • Add new design: Creates another Design View Sequence.
  • Insert a new design: Creates another Design View Sequence and inserts it at the selected location.
  • Load design from File: Allows the user to load a saved Design View sequence.
  • Cut / Copy / Paste: Allows the user to perform Cut, Copy, and Paste operations on the currently selected design sequence.
  • Remove Design: Removes the selected Design View Sequence from this view. A copy of the Design View Sequence is saved to file and can be loaded using the Load design from File function.
  • Remove All: Clears all the loaded Design View Sequence from this view.
  • Remove and Delete Sequence: Removes the selected Design View Sequence and deletes its associated file. Use the red dots to navigate between displayed design sequences.

 

Switching between Design Views

To toggle between the various Design Views that are currently open, left-click the red dots that appear below the Design View Sequence list (within the File Manager).
