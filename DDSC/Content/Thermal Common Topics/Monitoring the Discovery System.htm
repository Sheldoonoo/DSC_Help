Monitoring the Discovery System

The Discovery system may be monitored from either the instrument user interface or from the TRIOS software. See the help topic for a description of the integrated user interface display capability on your particular instrument.

TRIOS software provides a variety of ways to monitor the system.

  1. Control panel (Discovery DSC shown below): The Control panel provides a quick summary of the various components. The main Control panel provides the following :
  • Instrument State: Idle, Running, etc.
  • Current method segment and remaining run time (if running)
  • Numerical real-time signal display

Each component's control panel displays the current status and configuration information, as well as localized controls for the selected component.

  • General (provides temperature status)
  • Autosampler (provides current pan summary)
  • Running Method List (provides current method status)
  1. Active plot: Once an experiment is started, it can be easily monitored. The experiment document automatically displays data in spreadsheet and graph views.
  2. Status indicators: The status of Discovery system component is available through the status indicators at the bottom of the window display (see the image below). These indicators provide a quick overview of the current system. Below is a description of each of the indicators for the Discovery system.

  • Status: Indicates the current state of the instrument: Pre-test, Running, or Idle.
  • Sample Pan #: Indicates the current sample pan number in the cell when an experiment is in progress.
  • Temperature: Indicates the current temperature status of the monitored instrument.
  • Time: Displays the current time.
  1. Notifications Manager: The Notifications Manager contains all of the logged messages generated by the controller and by all connected instruments. Each log entry includes the Log Time, the type of message and an associated Code, a brief Description of what occurred, and a reference (Machine ID) indicating the source of the message.
